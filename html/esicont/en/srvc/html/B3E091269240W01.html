BASE REMOVAL

B3E091269240W01

1. Remove the rain sensor. (Vehicles with auto wiper system) (See RAIN SENSOR REMOVAL/INSTALLATION.)

2. Remove the rearview mirror. (See REARVIEW MIRROR REMOVAL/INSTALLATION.)

3. Wind each end of the SST (piano wire) around a bar.

Warning
• Using the SST (piano wire) with bare hands can cause injury. Always wear gloves when using the SST (piano wire).
Note
• Use a long sawing action to spread the work over the whole length of the SST (piano wire) to prevent it from breaking.

4. Saw through the sealant to remove the base.
