POWER WINDOW SYSTEM OUTLINE

B3E091258000T01

Features

• The following functions have been adopted for all windows.

- Manual open/close function
- Auto-open/close function
- Auto reverse pinch protection function
- Two-step down function
- Ignition off timer function
- Fail-safe function

• A power-cut function that permits disabling the operation of all window switches from the driver's seat has been adopted.

• The P/W CM (power window control module) inside the power window switch, which detects the door glass movement distance and direction based on pulse signals of the power window motor, controls these functions.
