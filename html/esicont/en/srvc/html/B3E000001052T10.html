DISCHARGE HEADLIGHT CONTROL MODULE FUNCTION

B3E000001052T10

• Controls the amount of electrical current while the discharge headlights are on to maintain optimum brightness together with lighting stability.

• The failure detection functions are as follows:

- Abnormal input detection function
- Abnormal output detection function

Abnormal Input Detection Function

• If the discharge headlight control module input voltage (9-16 V) fails to maintain operational voltage (except for the drop in voltage immediately after the headlights are turned on), the discharge headlight control module turns off the headlights for protection and to prevent partial operation.

• The discharge headlight control module turns the headlights back on at resumption of normal operational voltage.

Abnormal Output Detection Function

• If there is an abnormality in the output system (detects an open or GND short circuit in harness), the discharge headlight control module turns off the headlights for protection and to prevent partial operation errors.

• If the discharge headlight control module turns off the headlights due to an abnormality in the output system, the discharge headlight control module will maintain them in the off condition until the light switch is turned again from off to on.
