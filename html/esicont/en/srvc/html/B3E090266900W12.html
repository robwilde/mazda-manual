DTC 03:ER02

B3E090266900W12

DTC 03:Er02

CD player

DETECTION CONDITION

Note
• When error occurs, error message "CHECK CD" is displayed in the LCD.
• Cannot play CD normally or change tracks.

POSSIBLE CAUSE

• Defective CD (scratches or dirt)
• CD player malfunction

STEP

INSPECTION

ACTION

1
Play the CD and change tracks.
Can tracks be changed?
Yes
Go to Step 3.
No
Cannot change to specific tracks.
Go to the next step.
Cannot change to any track.
Replace the CD and try to change tracks again.
• If the function is normal, go to Step 3.
• If the function is not normal, replace the CD player.
2
Verify the condition of the CD when the error occurs.
Is the CD normal?
Yes
Go to the next step.
No
Repair (remove foreign material, clean) or replace the CD, and go to the next step.
3
Clear the DTC.
Play the CD and change tracks.
Is DTC 03: Er02 displayed?
Yes
Replace the CD player.
No
DTC troubleshooting completed.
