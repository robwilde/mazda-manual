• The generator warning light in the instrument cluster illuminates under the following conditions. (See DTC DETECTION LOGIC AND CONDITIONS [LF].)

- Charging system voltage problem
- Charging system voltage low
- Charging system voltage high
- IAT sensor circuit low input
- IAT sensor circuit high input
