Dust Seal Assembly Note

1. While rotating the piston counterclockwise using the SST, pull it out to the position shown in the figure.

2. As shown in the figure, assemble a new dust seal on the piston, and rotate the piston clockwise slowly using the SST to insert it to the caliper body completely.

3. Assemble the dust seal to the caliper body using the SSTs and a press with a press-in force of 834 N {85 kgf, 187 in·lbf}

4. Verify that there is no gap between the dust seal and caliper body.
