SPARK PLUG REMOVAL/INSTALLATION [ZJ, Z6]

B3E011801009W07

Caution
• If a spark plug that is not as specified is installed, sealing performance will be deteriorated. Install only the specified spark plug when replacing.

1. Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION [ZJ, Z6].)

2. Disconnect the negative battery cable.

3. Remove the air cleaner (L.H.D.). (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [ZJ, Z6].)

4. Set the air cleaner cover out of the way (R.H.D.).

5. Remove the air cleaner case (R.H.D.). (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [ZJ, Z6].)

6. Remove the ignition coil. (See IGNITION COIL REMOVAL/INSTALLATION [ZJ, Z6].)

7. Remove the spark plugs using a plug-wrench.

8. Install in the reverse order of removal.

Tightening torque
1422 N·m {1.52.3 kgf·m, 1116 ft·lbf}
