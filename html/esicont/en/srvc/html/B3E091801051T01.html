LIGHTING SYSTEMS OUTLINE

B3E091801051T01

• Built-in front turn and parking lights have been adopted for headlights.

• Discharge and halogen type headlights have been adopted.

• Projector type headlights (low-beam) have been adopted.

• Front fog lights have been adopted. (Located in front bumper)

• Stepped reflectors have been adopted for rear combination lights.

• LEDs have been adopted for the brake/taillights. (5HB)

• Ignition key illumination that illuminates the ignition key slot has been adopted.

• Interior light control system that changes illumination time and intensity using a keyless control module has been adopted.

• A headlight auto leveling system, which responds to the vehicle attitude and automatically adjusts the optical axis of the headlights, has been adopted.
