ON-BOARD DIAGNOSTIC SYSTEM ACTIVE COMMAND MODES FUNCTION (ABS, DYNAMIC STABILITY CONTROL)

B3E040243000T04

• The active command modes function is used for optionally selecting simulation items of input/output parts preset in the ABS HU/CM or DSC HU/CM, and to operate them regardless of CM control.

• To protect the hydraulic unit interior, operate output related parts for only 2 s or less when using the active command modes function.
