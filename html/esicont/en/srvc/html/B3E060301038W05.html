POOR POWER STEERING ASSIST

TROUBLESHOOTING HINTS
• Power steering fluid leakage from EHPAS fluid line
• Steering gear and linkage malfunction
• EHPAS control module malfunction

Diagnostic procedure

STEP

INSPECTION

ACTION

1
RETRIEVE DTC FROM EHPAS CONTROL MODULE
• Are any DTCs present?
Yes
Record all DTC and go to the applicable DTC inspection.
No
Go to the next step.
2
INSPECT STEERING WHEEL ASSIST FUNCTION
• Disconnect the EHPAS control module connector (2-pin).
• Is the power steering assist function changed?
Yes
Go to the next step.
No
Visually inspect the steering gear and linkage. Replace it if the malfunction is found.
3
INSPECT IF THERE IS ANY FLUID LEAK FROM THE EHPAS FLUID LINE
• Is there fluid leakage?
Yes
Replace it if necessary.
No
Replace the EHPAS control module.
(See ELECTRIC POWER STEERING OIL PUMP (LF) REMOVAL/INSTALLATION.)
