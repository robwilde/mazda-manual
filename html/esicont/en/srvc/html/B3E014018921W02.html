KNOCK SENSOR (KS) REMOVAL/INSTALLATION [LF]

B3E014018921W02

1. Remove the following part for easier access.

• Intake manifold (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [LF].)

2. Disconnect the KS connector.

3. Remove the KS installation bolt, then remove the KS.

4. Install in the reverse order of removal.

Tightening torque
16-24 N·m {1.7-2.4 kgf·m, 12-17 ft·lbf}
