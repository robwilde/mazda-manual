OBD FREEZE FRAME DATA [ZJ, Z6]

B3E010200102W15

• This is the technical data which indicates the engine condition at the time of the first malfunction. This data will remain in the memory even if another emission-related DTC is stored, with the exception of fuel system or misfire DTCs. Once freeze frame data for a fuel system or misfire DTC is stored, it will overwrite any previous data and the freeze frame will not be overwritten again.
