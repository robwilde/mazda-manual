PASSENGER JUNCTION BOX (PJB) OUTLINE

B3E094001072T01

• The following relays are built into the PJB:

- Headlight HI relay
- Head light LO relay
- TNS relay
- Rear fog light relay
- Windshield wiper relay
- Windshield wiper HI relay
- Rear wiper relay
- Door lock/unlock relays

• The PJB controls the following systems:

- Lighting system
- Wiper/washer system
- Power door lock system
- Keyless entry system
