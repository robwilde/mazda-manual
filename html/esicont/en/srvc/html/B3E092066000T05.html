COMPONENT PART AND FUNCTION

B3E092066000T05

Item
Function
Car-navigation unit
• Reads the data (map, voice and other) from the DVD-ROM.
• Calculates and displays vehicle position from various signals.
• Calculates the route to the destination.
• Navigates the driver to the destination using the map screen and/or voice.
Note
• DVD audio and video are not supported by this system.
• This unit does not support all Video CD and CD formats.
LCD unit
• Displays the screen (menus, maps and other screens) by remote control operation.
GPS antenna
• Receives GPS signal from satellites.
Gyro sensor (inside of the car-navigation unit)
• Sends yaw-rate signal to the CPU in the car-navigation unit.
TR switch (ATX) or back-up light switch (MTX)
• Sends R-range or reverse signal to the car-navigation unit.
DSC HU/CM or ABS HU/CM
• Sends vehicle speed signal to the car-navigation unit by CAN system.
Front door speaker and tweeter (driver side)
• Outputs voice and audio sound.
DVD-ROM (inside of the car-navigation unit)
• Contains map information data of each country.
• Contains voice data used to guide the route.
• Contains route information data to search for the route.
Car-navigation control switch
• Changes display screens, settings and other functions by button operation.
