Component and function

Variable valve timing actuator
• Continuously modifies the phases of the intake camshaft and crankshaft at the forward end of the intake camshaft using hydraulic pressure from the OCV.
OCV
• Operated by current (duty signal) from the PCM. Controls the hydraulic oil passages to the variable valve timing actuator.
CKP sensor
• Inputs engine revolution signal to the PCM.
CMP sensor
• Inputs cylinder identification signal to the PCM.
PCM
• Controls the OCV so that optimum valve timing is obtained according to engine operation conditions.
