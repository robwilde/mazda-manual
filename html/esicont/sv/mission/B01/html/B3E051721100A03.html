FUNKTION PÅ VÄXLINGSVENTILER A, B OCH C (TVÅNGSSTYRDA)

B3E051721100A03

• De tvångsstyrna växlingsventilerna ställer in trycket på utgångarna efter signal från PCM-modulen, och styr trycket på vardera koppling.

• Den tvångsstyrda växlingsventilen, som växlar PÅ/AV med 50 Hz (20 ms cykel) och styr det utgående trycket, är införd. Genom att ändra PÅ-tiden (0-100 %), kan magnetventilen anpassa tiden för ÖPPEN (inmatning) och STÄNGD (utmatning), för att upprätthålla kopplingstryck vid önskat hydraultryck. Därigenom kommer kopplingstrycket att öka när förhållandet (50 Hz PÅ förhållandet) minskar och sjunka när förhållandet ökar.
