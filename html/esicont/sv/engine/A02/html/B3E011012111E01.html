KONTROLL AV VENTILSPELEN

B3E011012111E01

1. Dra runt vevaxeln medsols så att kolv Nr1 befinner sig i kompressionsslagets övre dödpunkt.

2. Mät ventilspelen vid A enligt bilden.

• Om specifikationen inte uppfylls, byt ut ventillyftaren och justera ventilen till medianvärdet. (Se JUSTERING AV VENTILSPEL.)
Standard ventilspel [Kall motor]
0,27-0,33 mm {0,0107-0,0129 tum}
Observera
• Se till att notera de uppmätta värdena för att kunna bestämma lämpliga ersättningsshims.

3. Dra runt vevaxeln 360° medsols så att kolv Nr.4 befinner sig i kompressionsslagets övre dödpunkt.

4. Mät ventilspelen vid B i bilden.

• Om specifikationen inte uppfylls, byt ut ventillyftaren och justera ventilen till medianvärdet. (Se JUSTERING AV VENTILSPEL)
Standard ventilspel [Kall motor]
0,27-0,33 mm {0,0107-0,0129 tum}
Observera
• Se till att notera de uppmätta värdena för att kunna bestämma lämpliga ersättningsshims.
