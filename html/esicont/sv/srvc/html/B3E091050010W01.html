Observera vid demontering av främre stötfångare

1. Dra isär ändarna på stötfångaren (hjulhus) för att koppla loss den från fästet.

Försiktigt
• Den främre stötfångaren kan falla ner och skadas när man tar loss den från fästet. Säkra främre stötfångaren så den inte faller.

2. Tryck hållarna i pilens riktning.

3. Demontera främre stötfångaren från karossen.

Observera vid montering av främre stötfångare

1. Dra isär ändarna på stötfångaren.

2. Montera främre stötfångaren på karossen.

3. Tryck fast sidorna på stötfångaren i sina fästen på karossidan.
