INBYGGT DIAGNOSSYSTEM FRYSTA DATA [LF]

B3E010200102W04

• Detta är de tekniska data som visar motorns tillstånd vid tiden för det första felet. Dessa data sparas även om en annan avgasrelaterad felkod (DTC) sparas, med undantag för bränslesystemets eller misständningens felkoder (DTC). När frysta data för bränslesystemets eller misständningens felkoder (DTC) sparats, kommer tidigare data att skrivas över och dessa frysta data kommer inte att skrivas över igen.
