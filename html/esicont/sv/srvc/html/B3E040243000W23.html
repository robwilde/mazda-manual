DTC C1288, C1440 [DSC]

B3E040243000W23

Felkod (DTC)

C1288, C1440

Givare för bromsoljetryck

TILLSTÅND VID UPPTÄCKT

• C1288
- Spänningsvärdet från givaren för bromsoljetryck ligger inte inom specifikationen.
• C1440
- Utvärdet från givaren för bromsoljetryck ändras inte vid bromsning.
- Tryckvärdet från givaren för bromsoljetryck ligger inte inom specifikationen vid retardation.
- Tryckvärdet från givaren för bromsoljetryck överskrider specifikationen när man inte bromsar.

MÖJLIG ORSAK

• Fel i givaren för bromsoljetryck
• Avbrott eller kortslutning i kretsen för givaren för bromsoljetryck i DSC HU/CM
