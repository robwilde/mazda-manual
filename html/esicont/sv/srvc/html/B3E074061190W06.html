Luftblandningswire

1. Ställ in temperaturvredet på medelläge (MAX HOT-MAX COLD)

Försiktigt
• Om luftblandningsvajern fixeras utan att luftblandarlänkens läge fixerats kan det hända att luftblandningsklaffen inte kan röras normalt.

2. Ställ in luftblandningsmellanlänken sätt en diam. c:a ø5 Phillips-skruvmejsel i uttaget för att fixera länkens läge.

3. Anslut luftblandningsvajern till luftblandningslänken.

4. Kläm fast luftblandningsvajern i vajerklämman.

5. Demontera skruvmejseln och kontrollera att temperaturvredet kan vridas från ändstopp till ändstopp.

Luftströmsriktarvajer

1. Ställ in luftströmsriktarvredet på defroster.

Försiktigt
• Om luftströmsriktarvajern fixeras utan att huvudlänken för luftströmriktarens läge fixerats kan det hända att luftblandningsklaffen inte kan röras normalt.

2. Ställ in huvudlänken för luftströmriktaren på öppet läge och sätt en c:a ø5 Philips-skruvmejsel i uttaget för att fixera länkens läge.

3. Anslut luftströmsriktarvajern till luftströmsriktarlänken.

4. Kläm fast luftströmsriktarvajern i vajerklämman.

5. Ta bort skruvmejseln och kontrollera att luftströmsriktarvredet kan vridas från ändstopp till ändstopp.
