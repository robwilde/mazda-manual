DEMONTERING AV BASPLATTA

B3E091269240W01

1. Demontera regngivare. (För fordon med automatiska torkarblad) (Se DEMONTERING/MONTERING AV REGNGIVARE.)

2. Demontera den invändiga backspegeln. (Se DEMONTERING/MONTERING AV BACKSPEGEL.)

3. Linda specialverktygets(pianotrådens) ändar runt en stav.

Varni ng
• Att använda specialverktyget (pianotråden) med bara händer kan orsaka skador. Använd alltid skyddshandskar när du använder ett specialverktyg (pianotråd).
Observera
• Såga med långa tag för att sprida effekten över hela specialverktyget(pianotråd) för att förhindra att det går av.

4. Såga igenom tätningsmedlet för att demontera basplattan.
