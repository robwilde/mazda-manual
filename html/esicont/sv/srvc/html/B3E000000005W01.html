INSTALLATION AV BILRADIO

B3E000000005W01

• Om radion installerats fel eller en högeffektsmottagareanvänds kan CIS- och andra system påverkas. Observera följande vid installation av stereo i bilen:

- Montera antennen så långt ifrån kontrollmodulerna som möjligt.
- Montera antennmatarledningen så långt från kontrollmodulens kablage som möjligt.
- Se till att antennen och antennmatarledningen är justerade ordentligt.
- Installera inte högeffektmottagare.
