B3E071001038W02

Hantera för låg kylmedelsnivå

• Om otillräcklig kylmedelsnivå upptäcks vid felsökning, ladda inte (fyll inte på) kylmedel. Eftersom man inte kan avgöra vad som är en korrekt kylmedelsmängd från det tryck som visas på manometern, ladda inte kylmedlet. Om det finns för lite eller för mycket kylmedel efter påfyllningen kan det uppstå indirekta problem som skador på delarna i kylmedelssystemet, eller en minskning av kylprestanda. Töm därför ut kylmedlet helt ur kylmedelssystemet, om kylmedelsnivån är otillräcklig, och fyll åter på angiven mängd kylmedel.

Handhavande av kompressorolja

• Använd endast ATMOS GU10 kompressorolja till detta fordon. Användning av annan PAG-olja än ATMOS GU10 kompressorolja kan skada luftkonditioneringskompressorn.

• Spill inte ATMOS GU10 kompressorolja på fordonet. En droppe kompressorolja på fordonet kan förstöra lacken. Om det skulle komma kompressorolja på fordonet, torka omedelbart bort den.

• ATMOS GU10 kompressorolja (PAG-olja) har högre fuktupptagningsförmåga än den tidigare använda mineraloljan. Kylmedelssystemet kan skadas om fukt blandas med kompressoroljan. Därför måste locken sättas på omedelbart efter att kompressorolja har använts, eller så skall kylmedelsystemets delar avlägsnas för att förhindra fuktupptagning.

• Om kylmedlet försvinner helt ur systemet av olika orsaker som feltillstånd under A/C-drift, reparation eller byte av felaktig del, ladda kylmedel till angiven mängd och lägg alltid till 60 ml {60 cc, 2,03 fl oz} i kompressorn. Om man inte fyller på rätt mängd kompressorolja, kan A/C-kompressorn försämras snabbt, onormalt ljud kan uppstå, kylprestandan kan påverkas eller i värsta fall, kan A/C-kompressorn skära.
