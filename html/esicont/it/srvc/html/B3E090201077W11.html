SPIA LUMINOSA PROTEZIONE VEICOLO: 15

Il quadro strumenti rileva un numero ID della chiave non registrato.

DTC: B1601/P1260

CONDIZIONI

• Il quadro strumenti rileva un numero ID della chiave non registrato.

POSSIBILE CAUSA

• Dopo aver sostituito il quadro strumenti porte non è stata registrata nessuna chiave.
• Sono state usate chiavi non registrate
• Si è tentato di registrare una nona chiave
• Quadro strumenti malfunzionante

PASSO

ISPEZIONE

AZIONE

1
VERIFICARE IL NUMERO DI CHIAVI REGISTRATE
• Usando il WDS o uno strumento equivalente effettuare la procedura di controllo dati PID e verificare il numero di chiavi registrate.
(Vedere TABELLA CONTROLLO DATI PID (SISTEMA IMMOBILIZZATORE)).
• Risultano registrate una o più chiavi?
Sì

No

2
VERIFICARE IL NUMERO DI CHIAVI REGISTRATE
• Usando il WDS o uno strumento equivalente effettuare la procedura di controllo dati PID e verificare il numero di chiavi registrate.
(Vedere TABELLA CONTROLLO DATI PID (SISTEMA IMMOBILIZZATORE)).
• Risultano registrate otto chiavi?
Sì
Usando il WDS o uno strumento equivalente, cancellare tanti numeri ID delle chiavi quanti ne servono, quindi andare al passo successivo.
No
Andare al passo successivo.
3
ISPEZIONARE IL QUADRO STRUMENTI
• Registrare il numero ID della chiave tramite il WDS o uno strumento equivalente.
(Vedere SOSTITUZIONE COMPONENTI SISTEMA IMMOBILIZZATORE/DUPLICAZIONE E CANCELLAZIONE CHIAVI.)
Nota
• Per avviare il motore occorre registrare due o più chiavi.
• Usando la chiave registrata, mettere il commutatore d'avviamento in posizione ON.
