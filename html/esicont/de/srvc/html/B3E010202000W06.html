DTC P2006

Ansaugluft-Fallstromsteuerung hängt in Schließstellung

BEDINGUNG FÜR FEHLERERFASSUNG

• Das PCM überwacht die Ansaugluftmenge. Falls die tatsächliche Luftflussmenge die errechnete Luftflussmenge unterschreitet, wenn die folgenden Überwachungsbedingungen erfüllt sind. Erkennt das PCM, dass die variable Ansaugluft-Fallstromsteuerung in Schließstellung klemmt.
ÜBERWACHUNGSBEDINGUNG
- Die Kühlmitteltemperatur liegt über 70°C {158°F}.
- Die Drosselklappenöffnung liegt über 75%.
- die Motordrehzahl beträgt 4.500 U/min oder mehr. (ZJ)
- die Motordrehzahl beträgt 3.500 U/min oder mehr. (Z6)
Hinweise zur Diagnose
• Dies ist eine Dauerüberwachungsfunktion (CCM).
• Die Störungsanzeigeleuchte leuchtet auf, wenn das PCM die obige Störungsbedingung bei zwei aufeinanderfolgenden Testzyklen erfasst. Falls der Störungscode für die gleiche Störung im PCM gespeichert ist, spricht die Leuchte bereits beim ersten Testzyklus an.
• Ein VORGEMERKTER CODE wird registriert, wenn das PCM die obige Störungsbedingung bereits im ersten Testzyklus feststellt.
• RAHMENDATEN sind verfügbar.
• Der Störungscode wird im PCM-Speicher abgelegt.

MÖGLICHE URSACHE

• Luftmassenmesser defekt
• Nebenluft tritt ins Ansaugluftsystem ein (vom Luftmassenmesser an)
• Falsche Verlegung der Unterdruckschläuche
• VTCS-Stauklappe hängt in Schließstellung
• PCM defekt
