VORGEMERKTER OBD-DTC [LF]

B3E010200102W03

• Sie sprechen auf Mängel in den überwachten Systemen an. Beim ersten Störungsauftritt, d.h. während des ersten Testzykluses bzw. der ersten Testfahrt, speichert das PCM den Code, der dem gestörten System entspricht. Ein solcher Code wird als 'vorgemerkter Code' bezeichnet. Wenn das PCM feststellt, dass das System in den Normalzustand zurückgekehrt ist oder das Problem fälschlicherweise erkannt wurde, wird der vorgemerkte Code gelöscht. Sollte der Defekt jedoch auch bei der zweiten Fahrt auftreten, erkennt das PCM dies als echte Störung und speichert den vorgemerkten Code als Störungscode.
