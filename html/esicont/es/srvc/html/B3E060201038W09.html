DTC

C1278, C1956

Sensor ángulo dirección asistida

CONDICCION DE DETECCION

• C1278
- El módulo de control EHPAS detecta que la tensión de control del sensor ángulo dirección asistida es superior a 4,25 V o inferior a 0,5 V.
- El módulo de control EHPAS detecta que las dos tensiones de control modulan a la misma fase una cantidad especificada de fases o más.
- El módulo de control EHPAS detecta que la señal del sensor EHPAS no cambia.
• C1956
- El módulo de control EHPAS detecta que la tensión de alimentación del Sensor ángulo dirección asistida es superior a 5,5 V.
- El módulo de control EHPAS detecta que la tensión de alimentación del sensor ángulo dirección asistida es inferior a 4,5 V.

PROBABLE CAUSA

• Cortocircuito hacia alimentación, circuito abierto o cortocircuito hacia masa en el cableado entre el terminal 3B del módulo de control EHPAS y el terminal D del sensor ángulo dirección asistida
• Cortocircuito hacia alimentación, circuito abierto o cortocircuito hacia masa en el cableado entre el terminal 3D del módulo de control EHPAS y el terminal B del sensor ángulo dirección asistida
• Circuito abierto o cortocircuito hacia masa en el cableado entre el terminal 3C del módulo de control EHPAS y el terminal C del sensor ángulo dirección asistida
• Circuito abierto en el cableado entre el terminal 3A del módulo de control EHPAS y el terminal A del sensor ángulo dirección asistida
• Cortocircuito en el cableado entre los terminales B y D del sensor ángulo dirección asistida
• El anillo del sensor ángulo dirección asistida se ha caído.
• La tensión de alimentación del sensor ángulo dirección asistida es superior al valor especificado.
