DTC P0715

Funcionamiento incorrecto circuito sensor velocidad entrada/turbina

CONDICCION DE DETECCION

• Cuando haya satisfecho todas las condiciones y hayan pasados 0,7 segundos o más.
- Señal relaciones D o M del interruptor TR
- Conducir el vehículo a una velocidad de 40 km/h (25 mph) o superior.
- Señal del sensor velocidad entrada/turbina no activado
Detalles de soporte de diagnóstico:
• Esto es un control continuo (CCM).
• El MIL se enciende si el PCM detecta condiciones de funcionamiento incorrecto en el primer ciclo de conducción.
• El CODIGO PENDIENTE no es disponible.
• Las INSTANTANEAS MEMORIZADAS son disponibles.
• La luz de aviso AT se enciende.
• El DTC está registrado en la memoria del PCM.

PROBABLE CAUSA

• Funcionamiento incorrecto sensor velocidad entrada/turbina
• Cortocircuito hacia masa en el cableado entre el terminal A del sensor velocidad entrada/turbina y el terminal 1K (Z6)/1Q (LF) del PCM
• Cortocircuito hacia masa en el cableado entre el terminal B del sensor velocidad entrada/turbina y el terminal 1O (Z6)/1M (LF) del PCM
• Circuito abierto en el cableado entre el terminal A del sensor velocidad entrada/turbina y el terminal 1K (Z6)/1Q (LF) del PCM
• Circuito abierto en el cableado entre el terminal B del sensor velocidad entrada/turbina y el terminal 1O (Z6)/1M (LF) del PCM
• Conectores dañados entre el sensor velocidad entrada/turbina y el PCM
• Funcionamiento incorrecto del PCM
