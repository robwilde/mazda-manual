DTC P1260

Problema sistema inmovilizador

CONDICCION DE DETECCION

• El tablero de instrumentos detecta un funcionamiento incorrecto del sistema inmovilizador.
Detalles de soporte de diagnóstico
• Esto es un control continuo (Otros).
• El MIL no se enciende.
• El CODIGO PENDIENTE está disponible si el PCM detecta el funcionamiento incorrecto sobre mencionado.
• Los DATOS INSTANTANEOS MEMORIZADOS son disponibles.
• El DTC no está registrado en la memoria del PCM.

PROBABLE CAUSA

• Funcionamiento incorrecto del sistema inmovilizador
• Funcionamiento incorrecto del PCM

INSPECCION
