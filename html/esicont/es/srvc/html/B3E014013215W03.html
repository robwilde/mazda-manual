CONTROL SENSOR MASA FLUJO AIRE (MAF) (ZJ, Z6)

B3E014013215W03

Nota
• Antes de llevar a cabo el siguiente control, asegurarse de seguir el procedimiento indicado en el esquema de búsqueda averías. (Véase Procedimiento de búsqueda averías).

Control visual

1. Comprobar a vista el sensor MAF/IAT por lo siguiente:

• Daños, rupturas
• Terminal del sensor oxidado
• Terminal del sensor doblado
- En caso de funcionamiento incorrecto, sustituir el sensor MAF/IAT. [Véase REMOCION/INSTALACION SENSOR MASA FLUJO AIRE (MAF)/TEMPERATURA AIRE DE ADMISION (IAT) (ZJ, Z6)].

Control tensión

1. Quitar el sensor MAF/IAT sin desconectar el conector.

2. Poner el conmutador de arranque en posición ON.

3. Como el aire se aproxima gradualmente a la parte de detección MAF del sensor MAF/IAT, comprobar que la tensión de salida del sensor MAF (PID WDS: MAF) varíe.

• Si no se puede comprobar aunque el cableado no presente funcionamiento incorrecto, sustituir el sensor MAF/IAT. [Véase REMOCION/INSTALACION SENSOR MASA FLUJO AIRE (MAF)/TEMPERATURA AIRE DE ADMISION (IAT) (ZJ, Z6)].

Control circuitos abiertos/cortocircuitos

1. Quitar la tapa del conector del PCM.

2. Desconectar el conector del PCM. [Véase REMOCION/INSTALACION SISTEMA DE ADMISION (ZJ, Z6)].

3. Controlar por si hay circuito abierto o cortocircuito en los siguientes cableados (control continuidad).

Circuito abierto

• Si no hay continuidad, el circuito está abierto. Reparar o sustituir el cableado.
- Terminal A sensor MAF/IAT y terminal 2BG del PCM
- Terminal B sensor MAF/IAT y terminal 2BC del PCM
- Terminal C sensor MAF/IAT y terminal 2AU del PCM

Cortocircuito

• Si hay continuidad, hay un cortocircuito. Reparar o sustituir el cableado.
- Terminal A del sensor MAF/IAT y alimentación
- Terminal A del sensor MAF/IAT y masa carrocería
- Terminal B del sensor MAF/IAT y alimentación
- Terminal C del sensor MAF/IAT y alimentación
- Terminal C del sensor MAF/IAT y masa carrocería
