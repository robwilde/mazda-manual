DTC

C2785

Sistema sensor combinado (procedimiento de inicialización no efectuado)

CONDICCION DE DETECCION

• El DSC detecta que el procedimiento de inicialización para el sensor combinado no fue efectuado.

PROBABLE CAUSA

• Procedimiento de inicialización para el sensor combinado no efectuado
• Funcionamiento incorrecto sensor combinado
