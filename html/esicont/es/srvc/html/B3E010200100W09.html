B3E010200100W09

DTC P0122

Entrada baja circuito sensor TP

CONDICCION DE DETECCION

• Si el PCM detecta que la tensión del sensor TP en el terminal 2AA del PCM es inferior a 0,10 V con motor en marcha, el PCM determina que el circuito del sensor TP no funciona correctamente.
Detalles de soporte de diagnóstico
• Esto es un control continuo (CCM).
• El MIL se enciende si el PCM detecta condiciones de funcionamiento incorrecto en el primer ciclo de conducción.
• El CODIGO PENDIENTE está disponible si el PCM detecta el funcionamiento incorrecto sobre mencionado.
• Las INSTANTANEAS MEMORIZADAS son disponibles.
• El DTC está registrado en la memoria del PCM.

PROBABLE CAUSA

• Funcionamiento incorrecto del sensor TP
• Funcionamiento incorrecto del conector o del terminal
• Circuito abierto en el cableado entre el terminal A del sensor TP y el terminal 2W del PCM
• Cortocircuito hacia masa en el cableado entre el terminal A del sensor TP y el terminal 2W del PCM
• Circuito abierto en el cableado entre el terminal C del sensor TP y el terminal 2AA del PCM
• Cortocircuito hacia masa en el cableado entre el terminal C del sensor TP y el terminal 2AA del PCM
• Funcionamiento incorrecto del PCM
