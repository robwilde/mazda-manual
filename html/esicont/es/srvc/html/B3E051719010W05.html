CONTROL SENSOR DE VELOCIDAD VEHICULO (VSS)

B3E051719010W05

1. Conectar el WDS o instrumento equivalente, conducir el vehículo y controlar el valor de entrada del PID OSS.

• Si el valor de entrada es 0 rpm, efectuar el 'Control tensión de alimentación', el 'Control circuitos abiertos' o el 'Control cortocircuitos'.
• Si el valor de entrada no es 0 rpm, efectuar el 'Control visual' o el 'Control del perfil de la onda'.

Control visual

1. Quitar el VSS. [Véase REMOCION/INSTALACION SENSOR VELOCIDAD VEHICULO (VSS)].

2. Comprobar que el sensor no lleve partículas o virutas metálicas.

• En caso de funcionamiento incorrecto, limpiarlas.

3. Instalar el VSS. [Véase REMOCION/INSTALACION SENSOR VELOCIDAD VEHICULO (VSS)].

Control del perfil de la onda

1. Quitar el PCM. [Véase REMOCION/INSTALACION SISTEMA DE ADMISION (ZJ, Z6)]. [Véase REMOCION/INSTALACION BATERIA (LF)].

2. Conectar el WDS o instrumento equivalente al DLC-2.

3. Conectar los cables de prueba del osciloscopio a los siguientes terminales del conector del PCM.

•  Cable (+): terminal 1AY del PCM
•  (cable (-) : terminal 2Y del PCM:

4. Poner en marcha el motor.

5. Controlar el PID VSS.

6. Controlar el perfil de la onda.

• Terminal del PCM: 1AY (+)-2Y (-)
• Ajuste del osciloscopio: 1 V/DIV (Y), 2,5 ms/DIV (X), gama CC
• Condiciones del vehículo: conducir el vehículo a una velocidad de 32 km/h (20 mph)
- En caso de funcionamiento incorrecto, efectuar el 'Control circuitos abiertos' o el 'Control cortocircuitos'

Control tensión de alimentación

1. Desconectar el conector del VSS.

2. Poner el conmutador de arranque en posición ON.

3. Medir la tensión en el terminal B del VSS.

Especificación
4,5-5,5 V
• Si la tensión es correcta, pasar al 'Control circuitos abiertos' y al 'Control cortocircuitos'.
• En caso de funcionamiento incorrecto, reparar el cableado entre el sensor VSS y el PCM.

Control circuito abierto

1. Controlar si los siguientes circuitos están abiertos.

• Circuito de alimentación (del terminal A del sensor VSS al terminal D del relé principal)
• Circuito de masa (del terminal C del sensor VSS a la masa)
• Si se encuentra un circuito abierto o un cortocircuito, reparar el cableado que no funciona correctamente.
• Si no hay circuitos abiertos o cortocircuitos, llevar a cabo el control del rotor del sensor.

Control cortocircuito

1. Controlar si los siguientes circuitos están en cortocircuito.

• Circuito de alimentación (del terminal A del sensor VSS al terminal D del relé principal)
• Si se encuentra un circuito abierto o un cortocircuito, reparar el cableado que no funciona correctamente.
• Si no hay circuitos abiertos o cortocircuitos, llevar a cabo el control del rotor del sensor.

Control rotor sensor

1. Quitar el VSS. [Véase REMOCION/INSTALACION SENSOR VELOCIDAD VEHICULO (VSS)].

2. Poner la palanca selector en posición N.

3. Controlar la superficie del rotor del sensor por media de la tapa de instalación del VSS mientras se rueda manualmente el neumático delantero.

(1) ¿El rotor del sensor no lleva daños y grietas?
(2) ¿El rotor del sensor no lleva partículas o virutas metálicas?
• Si el rotor del sensor no está dañado, sustituir el sensor VSS.
• En caso de funcionamiento incorrecto, limpiar o sustituir el rotor del sensor.
