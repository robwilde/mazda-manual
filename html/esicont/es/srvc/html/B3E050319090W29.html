N.25 EL MOTOR SE CALA CUANDO SE VIAJA A BAJA VELOCIDAD O CUANDO SE PARA EL VEHICULO

B3E050319090W29

25

El motor se cala cuando se viaja a baja velocidad o cuando se para el vehículo

DESCRIPCION

• El motor se cala cuando se pisa el pedal del freno mientras se viaja a baja velocidad o cuando se para el vehículo.

PROBABLE CAUSA

• El funcionamiento incorrecto está en el sistema de control motor (por ejemplo control inyección combustible, sistema IAC). De lo contrario, el funcionamiento incorrecto está en el cuerpo de la válvula reguladora, electroválvula de cambio relación E o TCC.
