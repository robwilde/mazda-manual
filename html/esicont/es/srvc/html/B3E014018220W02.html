CONTROL SENSOR POSICION CIGUENAL (CKP) (ZJ, Z6)

B3E014018220W02

Nota
• Antes de llevar a cabo el siguiente control, asegurarse de seguir el procedimiento indicado en el esquema de búsqueda averías. (Véase Procedimiento de búsqueda averías).

Control visual

1. Quitar el sensor CKP. [Véase REMOCION/INSTALACION SENSOR POSICION CIGUENAL (CKP) (ZJ, Z6)].

2. Comprobar que no haya virutas de metal en el sensor.

3. Instalar el sensor CKP. [Véase REMOCION/INSTALACION SENSOR POSICION CIGUENAL (CKP) (ZJ, Z6)].

Control del entrehierro

1. Comprobar que el sensor CKP esté instalado firmemente.

2. Con un calibre de espesores, medir el entrehierro entre los salientes de la placa en la parte trasera de la polea de cigüeñal y el sensor CKP.

• Si no está conforme a lo especificado, controlar por si los salientes de la placa están grietados o doblados.
- Si no está conforme a lo especificado, sustituir la placa. [Véase SUSTITUCION SELLO DE ACEITE DELANTERO (ZJ, Z6)].
Distancia entre los electrodos
0,5-1,5 mm (0,02-0,05 in)

Control tensión

1. Poner el motor al mínimo.

Advertencia
• Si el agua entra en el conector, causa el funcionamiento incorrecto del sensor. Para evitar ésto, tener cuidado de no dañar los cableados o el conector impermeable que impide la entrada del agua.

2. Medir la tensión de salida con un osciloscopio.

• Si no está conforme a lo especificado, sustituir el sensor CKP. [Véase REMOCION/INSTALACION SENSOR POSICION CIGUENAL (CKP) (ZJ, Z6)].
• Si la condición/especificación (referencia) de los elementos de control no está conforme a lo especificado, incluso si no hay funcionamiento incorrecto, llevar a cabo el "Control circuitos abiertos/cortocircuitos".

Control circuitos abiertos/cortocircuitos

1. Quitar la tapa del conector del PCM.

2. Desconectar el conector del PCM. [Véase REMOCION/INSTALACION SISTEMA DE ADMISION (ZJ, Z6)].

3. Controlar por si hay circuito abierto o cortocircuito en los siguientes cableados (control continuidad).

Circuito abierto

• Si no hay continuidad, el circuito está abierto. Reparar o sustituir el cableado.
- Terminal A del sensor CKP y terminal 2T del PCM
- Terminal B del sensor CKP y terminal 2P del PCM
- Terminal C del sensor CKP y terminal 2BF del PCM

Cortocircuito

• Si hay continuidad, hay un cortocircuito. Reparar o sustituir el cableado.
- Terminal A del sensor CKP y masa carrocería
- Terminal B del sensor CKP y alimentación
- Terminal B del sensor CKP y masa carrocería
- Terminal C del sensor CKP y alimentación
