DTC B1231

Congelación control activación (despliegue) del módulo de control SAS

CONDICCION DE DETECCION

Atención
• Las condiciones de detección sirven para comprender el DTC antes de efectuar un control. Efectuar un control sólo con las condiciones de detección puede causar lesiones debidas a un error de funcionamiento o daños al sistema. Cuando se efectue un control, seguir siempre el procedimiento de control.
• El módulo de control SAS ha determinado que ocurrió una colisión

Procedimiento de diagnóstico

ACCION

Sustituir el módulo de control SAS.
(Véase REMOCION/INSTALACION MODULO DE CONTROL SAS).
