SISTEMA DE COMUNICACION MULTIPLEX

B3E090255430W01

Generalidades

• Si el sistema red de área controlada (CAN) no funciona correctamente, leer los DTC de los siguientes módulos, utilizando el WDS o instrumento equivalente para determinar cuál es el sistema que no funciona correctamente.

- PCM
- Módulo de control EHPAS
- DSC HU/CM (con DSC)
- ABS HU/CM (con ABS)
- Tablero de instrumentos
- Unidad de control climatización
- Módulo de control SAS
- Unidad audio (módulo base)
- PJB
