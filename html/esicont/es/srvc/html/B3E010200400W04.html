B3E010200400W04

DTC P0480

Problema circuito de control ventilador de refrigeración

CONDICCION DE DETECCION

• El PCM controla la tensión de entrada del módulo de control ventilador. Si la tensión en el terminal 1AP del PCM se queda baja o alta, el PCM determina que hay un problema en el circuito de control del ventilador de refrigeración.
Detalles de soporte de diagnóstico
• Esto es un control continuo (Otros).
• El MIL no se enciende.
• Los DATOS PID INSTANTANEOS MEMORIZADOS no son disponibles.
• El DTC está registrado en la memoria del PCM.

PROBABLE CAUSA

• Funcionamiento incorrecto módulo de control ventilador
• Funcionamiento incorrecto del conector o del terminal
• Circuito abierto en el cableado entre el terminal C del módulo de control ventilador y el terminal 1AP del PCM
• Cortocircuito hacia alimentación en el cableado entre el terminal C del módulo de control ventilador y el terminal 1AP del PCM
• Cortocircuito hacia masa en el cableado entre el terminal C del módulo de control ventilador y el terminal 1AP del PCM
• Funcionamiento incorrecto del PCM
