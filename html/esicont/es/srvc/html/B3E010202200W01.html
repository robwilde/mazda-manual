DTC P2228 [ZJ, Z6]

B3E010202200W01

DTC P2228

Entrada baja circuito sensor BARO

CONDICCION DE DETECCION

• El PCM controla la tensión de entrada del sensor BARO. Si la tensión de entrada en el terminal 2S del PCM es inferior a 1,7V, el PCM establece que el circuito del sensor BARO no funciona correctamente.
Detalles de soporte de diagnóstico
• Esto es un control continuo (CCM).
• El MIL se enciende si el PCM detecta condiciones de funcionamiento incorrecto en el primer ciclo de conducción.
• El CODIGO PENDIENTE está disponible si el PCM detecta el funcionamiento incorrecto sobre mencionado.
• Las INSTANTANEAS MEMORIZADAS son disponibles.
• El DTC está registrado en la memoria del PCM.

PROBABLE CAUSA

• Funcionamiento incorrecto del sensor BARO.
• Funcionamiento incorrecto del conector o del terminal
• Circuito abierto en el cableado entre el terminal C del sensor BARO y el terminal 2W del PCM
• Cortocircuito hacia masa en el cableado entre el terminal C del sensor BARO y el terminal 2W del PCM
• Circuito abierto en el cableado entre el terminal B del sensor BARO y el terminal 2S del PCM
• Cortocircuito hacia masa en el cableado entre el terminal B del sensor BARO y el terminal 2S del PCM
• Funcionamiento incorrecto del PCM
