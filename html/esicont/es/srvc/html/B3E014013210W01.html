CONTROL SENSOR MASA FLUJO AIRE (MAF) (LF)

B3E014013210W01

Nota
• Antes de llevar a cabo el siguiente control, asegurarse de seguir el procedimiento indicado en el esquema de búsqueda averías.

Control visual

1. Comprobar a vista el sensor MAF/IAT por lo siguiente:

• Daños, rupturas
• Terminal del sensor oxidado
• Terminal del sensor doblado
- En caso de funcionamiento incorrecto, sustituir el sensor MAF.
- Si la condición/especificación (referencia) de los elementos de control no está conforme a lo especificado, incluso si no hay funcionamiento incorrecto, llevar a cabo el "Control circuitos abiertos/cortocircuitos".

Control tensión

1. Quitar el sensor MAF/IAT sin desconectar el conector.

2. Poner el conmutador de arranque en posición ON.

3. Como el aire se aproxima gradualmente a la parte de detección MAF del sensor MAF/IAT, comprobar que la tensión del terminal 1AC del PCM (PID WDS: MAF) varíe.

• Si no se puede comprobar incluso aunque el cableado no presente funcionamiento incorrecto, sustituir el sensor MAF/IAT.

Control circuitos abiertos/cortocircuitos

1. Desconectar el conector del PCM. [Véase REMOCION/INSTALACION PCM (LF)].

2. Desconectar el conector del sensor MAF.

3. Controlar si los siguientes cableados están abiertos o en cortocircuito. (Control de la continuidad)

Circuito abierto

• Si no hay continuidad, el circuito está abierto. Reparar o sustituir el cableado.
- Terminal A del sensor MAF y terminal E del relé principal
- Terminal B del sensor MAF y terminal 1BD del PCM
- Terminal C del sensor MAF y terminal 1AC del PCM

Cortocircuito

• Si hay continuidad, el circuito está en cortocircuito. Reparar o sustituir el cableado.
- Terminal A del sensor MAF y masa carrocería
- Terminal C del sensor MAF y alimentación
- Terminal C del sensor MAF y masa carrocería
- Terminal B del sensor MAF y alimentación
