CONDICCION DE DETECCION

Atención
• Las condiciones de detección sirven para comprender el DTC antes de efectuar un control. Efectuar una inspección sólo de acuerdo con las condiciones de detección puede causar lesiones debidas a un error de funcionamiento o daños al sistema. Cuando se efectue un control, seguir siempre el procedimiento de control.
• El módulo de control SAS detecta un error de la magnitud del impacto ajustada en el sensor, que ocurre debido a una instalación incorrecta del sensor de colisióno de los sensores airbag laterales.

PROBABLE CAUSA

• Sensor de colisión incorrectamente instalado con el sensor equivocado
• Sensor airbag lateral lado conductor incorrectamente instalado con el sensor equivocado
• Sensor airbag lateral lado pasajero incorrectamente instalado con el sensor equivocado
