MINIMO BAJO/EL MOTOR SE CALA DURANTE LA DESACELERACION

DESCRIPCION

• El motor se para de improviso al empezar una desaceleración o una aceleración después de una desaceleración.

PROBABLE CAUSA

• Pérdida de vacío
• Funcionamiento incorrecto válvula IAC
• Pérdidas de aire del sistema de admisión aire
• Regulación relación mezcla aire/combustible incorrecta
• Funcionamiento incorrecto sistema de control emisiones de vapor
• Ajuste incorrecto sensor TP
• Funcionamiento incorrecto sensor TP o circuito relacionado
• Funcionamiento incorrecto sensor MAP o circuito relacionado
• Funcionamiento incorrecto interruptor frenos o circuito relacionado
• Funcionamiento incorrecto interruptor posición pedal del embrague o circuito relacionado (MTX)
• Funcionamiento incorrecto interruptor TP o circuito relacionado (ATX)
• Funcionamiento inmcorrecto embrague magnético A/C
