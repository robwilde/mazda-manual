DTC 06:Er02

Cargador CD (módulo superior)

CONDICCION DE DETECCION

Nota
• Cuando se verifique el error, en la pantalla LCD se visualiza 'CHECK CD'.
• No se puede escuchar el CD normalmente o no se puede cambiar el pasaje.

PROBABLE CAUSA

• CD defectuoso (rayado o sucio)
• Funcionamiento incorrecto Cargador CD (módulo superior)
