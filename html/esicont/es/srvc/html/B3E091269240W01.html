REMOCION DE LA BASE

B3E091269240W01

1. Quitar el sensor lluvia. (vehículos con sistema limpiaparabrisas automático) (VéaseREMOCION/INSTALACION SENSOR LLUVIA).

2. Quitar el espejo retrovisor interior. (Véase REMOCION/INSTALACION ESPEJO RETROVISOR INTERIOR).

3. Envolver cada extremidad de la SST(cuerda de piano) alrededor de una barra.

Atención
• Utilizar la SST (cuerda de piano) con las manos desnudas puede causar heridas. Llevar siempre unos guantes cuando se utilice una SST (cuerda de piano).
Nota
• Para evitar que la SST (cuerda de piano) se rompa, efectuar el corte disfrutando toda la largura del alambre.

4. Cortar la capa de masilla impermeable para quitar la base.
