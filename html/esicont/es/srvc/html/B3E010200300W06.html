B3E010200300W06

DTC P0340

Problema circuito sensor CMP

CONDICCION DE DETECCION

• El PCM controla la tensión de entrada del sensor CMP con motor en marcha. Si el PCM no recibe tensión de entrada del sensor CMP durante 12 rotaciones consecutivas del motor, el PCM determina que hay un problema en el circuito del sensor CKP.
Detalles de soporte de diagnóstico
• Esto es un control continuo (CCM).
• El MIL se enciende si el PCM detecta condiciones de funcionamiento incorrecto en el primer ciclo de conducción.
• El CODIGO PENDIENTE está disponible si el PCM detecta el funcionamiento incorrecto sobre mencionado.
• Las INSTANTANEAS MEMORIZADAS son disponibles.
• El DTC está registrado en la memoria del PCM.

PROBABLE CAUSA

• Funcionamiento incorrecto sensor CMP
• Funcionamiento incorrecto del conector o del terminal
• Circuito abierto en el cableado entre el terminal A del sensor CMP y el terminal 2T del PCM
• Cortocircuito hacia masa en el cableado entre el terminal A del sensor CMP y el terminal 2T del PCM
• Circuito abierto en el cableado entre el terminal B del sensor CMP y el terminal 2BB del PCM
• Cortocircuito hacia alimentación en el cableado entre el terminal B del sensor CMP y el terminal 2BB del PCM
• Cortocircuito hacia masa en el cableado entre el terminal B del sensor CMP y el terminal 2BB del PCM
• Circuito abierto en el cableado entre el terminal C del sensor CMP y el terminal 2BF del PCM
• Cortocircuito hacia alimentación en el cableado entre el terminal C del sensor CMP y el terminal 2BF del PCM
• Funcionamiento incorrecto del PCM
