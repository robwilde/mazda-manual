CONTROL DEL SENSOR DE POSICION MARIPOSA (TP) (ZJ, Z6)

B3E014018910W04

Nota
• Antes de llevar a cabo el siguiente control, asegurarse de seguir el procedimiento indicado en el esquema de búsqueda averías. (Véase Procedimiento de búsqueda averías).

Control de la resistencia

1. Controlar lo siguiente.

• Condición de válvula de mariposa cerrada
• Juego cable del acelerador [Véase CONTROL/AJUSTE CABLE DEL ACELERADOR (ZJ, Z6)]

2. Desconectar el conector del sensor TP.

3. Comprobar que la resistencia entre los terminales B y C cambie de forma moderada según la apertura de la válvula de mariposa.

• Si la resistencia cambia, pasar al paso siguiente.
• Si la resistencia no cambia, sustituir el sensor TP. [Véase REMOCION/INSTALACION SENSOR DE POSICION MARIPOSA (TP) (ZJ, Z6)].

4. Medir la resistencia entre los terminales A y B.

• Si no está conforme a lo especificado, sustituir el sensor TP. [Véase REMOCION/INSTALACION SENSOR DE POSICION MARIPOSA (TP) (ZJ, Z6)].
• Si la condición/especificación (referencia) de los elementos de control no está conforme a lo especificado, incluso si no hay funcionamiento incorrecto, llevar a cabo el "Control circuitos abiertos/cortocircuitos".
Resistencia sensor TP
2,5-6,0 kilohm [25 °C (77 °F)]

Control circuitos abiertos/cortocircuitos

1. Quitar la tapa del conector del PCM.

2. Desconectar el conector del PCM. [Véase REMOCION/INSTALACION SISTEMA DE ADMISION (ZJ, Z6)].

3. Controlar por si hay circuito abierto o cortocircuito en los siguientes cableados (control continuidad).

Circuito abierto

• Si no hay continuidad, el circuito está abierto. Reparar o sustituir el cableado.
- Terminal A del sensor TP y terminal 2W del PCM
- Terminal B del sensor TP y terminal 2AE del PCM
- Terminal A del sensor TP y terminal 2AA del PCM

Cortocircuito

• Si hay continuidad, hay un cortocircuito. Reparar o sustituir el cableado.
- Terminal A del sensor TP y masa carrocería
- Terminal B del sensor TP y alimentación
- Terminal C del sensor TP y alimentación
- Terminal C del sensor TP y masa carrocería
