Nota
• Cuando se giran sólo las ruedas motrices mientras el vehículo está levantado, los DTC C1235 y C1236 se memorizan.

DTC

C1141, C1142, C1143, C1144, C1233, C1234, C1235, C1236

Sensor velocidad rueda ABS/rotor del sensor ABS

CONDICCION DE DETECCION

• C1141, C1142, C1143, C1144
- Se detecta una anomalía periódica en el perfil de la onda de los sensores velocidad rueda ABS.
• C1234, C1233, C1235, C1236
- La señal de velocidad de una de las cuatro ruedas no entra o es muy débil cuando se conduce a una velocidad de 10 km/h (6,2 mph) o superior.
- Se detecta un grande, repentino cambio en la señal de velocidad de la rueda.
- El control ABS funciona durante 28 segundos o más.

PROBABLE CAUSA

• Funcionamiento incorrecto sensor velocidad rueda ABS
• Funcionamiento incorrecto rotor del sensor ABS (presencia de material extraño)
• Instalación incorrecta sensor velocidad rueda ABS y/o rotor del sensor
• Juego excesivo entre el sensor velocidad rueda ABS y el rotor del sensor
• Funcionamiento continuo ABS
