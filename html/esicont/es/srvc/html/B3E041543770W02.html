CONTROL SENSOR COMBINADO

B3E041543770W02

1. Poner el conmutador de arranque en posición OFF.

2. Conectar el WDS o instrumento equivalente al DLC-2.

3. Seleccionar los siguientes PID, y luego controlar la velocidad de aceleración lateral y el derrape.

- LAT_ACCL (aceleración lateral)
- YAW_RATE (derrape)
(1) Control aceleración lateral
1) Comprobar el cambio LAT_ACCL cuando el sensor combinado se inclina a la izquierda y a la derecha.

• En caso de funcionamiento incorrecto, sustituir el sensor combinado. (Véase REMOCION/INSTALACION SENSOR COMBINADO).
Estándar
Cuando el sensor se inclina hacia la derecha (A):
LAT_ACCL cambia en positivo.
Cuando el sensor se inclina hacia la izquierda (B):
LAT_ACCL cambia en negativo.
(2) Control de derrape
1) Comprobar el cambio YAW_RATE cuando el sensor combinado gira a la izquierda y a la derecha.

• En caso de funcionamiento incorrecto, sustituir el sensor combinado. (Véase REMOCION/INSTALACION SENSOR COMBINADO).
Estándar
Cuando el sensor se gira hacia la derecha (A):
YAW_RATE cambia positivamente.
Cuando el sensor se gira hacia la izquierda (B):
YAW_RATE cambia negativamente.
