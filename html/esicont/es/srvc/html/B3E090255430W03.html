Procedimientos de reparación

1. Controlar el conector del módulo defectuoso.

• Si hay funcionamiento incorrecto, sustituir o reparar el conector.

2. Controlar el cableado defectuoso como sigue:

• En caso de funcionamiento incorrecto, reparar o sustituir el cableado.
• En caso de funcionamiento incorrecto, sustituir el módulo defectuoso.
- Cortocircuito hacia masa
- Cortocircuito hacia alimentación
- Cables acoplados en cortocircuito entre si
- Circuito abierto

3. Volver a conectar todos los conectores desconectados.

4. Borrar los DTC relacionados con el sistema CAN mediante el WDS o instrumento equivalente.

5. Borrar los DTC relacionados con el sistema CAN mediante el WDS o instrumento equivalente.

• Si aparecen de nuevo los mismos DTC, sustituir el módulo defectuoso.
- U0073 (PCM, módulo de control EHPAS, tablero de instrumentos, módulo de control SAS)
- U0516 (unidad de climatización)
- U1900 (PJB)
- U2012 (DSC HU/CM o ABS HU/CM)
- U2516 (tablero de instrumentos)
• Si aparece otro DTC, efectuar el control del DTC apropiado. [Véase TABLA DE LOS DTC (SISTEMA DE COMUNICACION MULTIPLEX)].
