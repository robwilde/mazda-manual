Síntoma

3

Vehículos con DSC

N.

Síntoma

8
Cualquiera de las siguientes luces no se iluminan cuando el conmutador de arranque se cambia a la posición ON.
• Luz de aviso ABS
• Luz de aviso sistema FRENOS
• Indicador visual DSC
• Indicador visual DSC OFF
9
Cualquiera de las siguientes luces se queda en ON:
• Luz de aviso ABS
• Luz de aviso sistema FRENOS
• Indicador visual DSC
• Indicador visual DSC OFF
10
Hay un funcionamiento incorrecto en el sistema aún si la luz de aviso ABS, la luz de aviso sistema frenos, el indicador visual DSC y el indicador visual DSC OFF no se encienden.
11
ABS o TCS*1 frecuentemente en función.
El TCS no funciona correctamente.
12
El DSC*2 frecuentemente en función.
El DSC no funciona correctamente.

*1 :
El sistema DSC contiene la función de control de la tracción; el indicador visual DSC se enciende y apaga cuando funciona el DSC.
*2 :
El indicador visual DSC se enciende y apaga cuando funciona el DSC.
