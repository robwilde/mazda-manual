CONTROL SENSOR PRESION BAROMETRICA (BARO) (LF)

B3E014018210W05

Nota
• Antes de llevar a cabo el siguiente control, asegurarse de seguir el procedimiento indicado en el esquema de búsqueda averías.

Control tensión

1. Quitar el sensor BARO con el conector todavía conectado.

2. Poner el conmutador de arranque en posición ON.

3. Comprobar que la tensión en el terminal 1AG del PCM (PID WDS: BARO) esté conforme a lo especificado.

• Si no está conforme a lo especificado, aunque el cableado no presente funcionamiento incorrecto, sustituir el sensor BARO.
Tensión de salida del sensor BARO
2,3-4,7 V

4. Desconectar el tubo flexible del sensor BARO.

5. Instalar la bomba de vacío al sensor BARO.

6. Aplicar un vacío de 30 kPa al sensor BARO y comprobar que el cambio de tensión en el terminal 1AG del PCM (PID WDS: BARO) esté conforme a lo especificado.

• Si no está conforme a lo especificado, sustituir el sensor BARO.
Variación tensión de salida sensor BARO
0,97-1,41 V

Control circuitos abiertos/cortocircuitos

1. Desconectar el conector del PCM. [Véase REMOCION/INSTALACION PCM (LF)].

2. Controlar si los siguientes cableados están abiertos o en cortocircuito. (Control de la continuidad)

Circuito abierto

• Si no hay continuidad, el circuito está abierto. Reparar o sustituir el cableado.
- Entre el terminal A del sensor presión barométrica y el terminal 1AG del PCM
- Terminal B del sensor presión barométrica y terminal 1AA del PCM
- Terminal C del sensor presión barométrica y terminal 1AE del PCM

Cortocircuito

• Si hay continuidad, el circuito está en cortocircuito. Reparar o sustituir el cableado.
- Terminal C del sensor presión barométrica y alimentación
- Terminal C del sensor presión barométrica y masa carrocería
- Terminal A del sensor presión barométrica y alimentación
- Terminal A del sensor presión barométrica y masa carrocería
