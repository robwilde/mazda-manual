Reproductor de casetes-Línea de comunicación unidad de base

CONDICCION DE DETECCION

• La unidad de base detecta un error de comunicación con el reproductor de casetes.

PROBABLE CAUSA

• Funcionamiento incorrecto en los conectores entre la unidad de base y el reproductor de casetes
• Funcionamiento incorrecto del reproductor de casetes
• Funcionamiento incorrecto unidad de base
