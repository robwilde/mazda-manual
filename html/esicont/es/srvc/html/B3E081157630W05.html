CONTROL CINTURONES DE SEGURIDAD

B3E081157630W05

Cinturón

1. Comprobar que el cinturón se instale adecuadamente, que no se tuerca ni se enrosque.

2. Comprobar que el cinturón de seguridad no tiene daños y no está deformado por los empalmes de metal.

• En caso de funcionamiento incorrecto, sustituir el cinturón de seguridad.

ELR

1. Comprobar que los cinturones de seguridad se envuelvan y desenvuelvan regularmente.

• En caso de funcionamiento incorrecto, sustituir el cinturón de seguridad.

2. Controlar que el enrollador se bloquee cuando el cinturón se extrae rápidamente.

• En caso de funcionamiento incorrecto, sustituir el cinturón de seguridad.

3. Quitar el enrollador.

4. Cuando se tire del cinturón de seguridad, asegurarse de que no se bloquee cuando se inclina el enrollador unos 15° con respeto a su posición de instalación y que se bloquee cuando se inclina de 40° o más.

• En caso de funcionamiento incorrecto, sustituir el cinturón de seguridad.

ALR

1. Extrayendo completamente el cinturón de seguridad el enrollador pasa de la modalidad ELR (enrollador de emergencia) a la modalidad ALR (enrollador automático).

2. Comprobar que se oiga el clic del enrollador mientras que el cinturón se enrolla lentamente. Si no se oye el clic, la modalidad ALR no se ha activado. Si necesario, repetir el paso 1.

• En caso de funcionamiento incorrecto, sustituir el cinturón de seguridad.

3. Comprobar que el enrollador se bloquee cuando se tira.

• En caso de funcionamiento incorrecto, sustituir el cinturón de seguridad.

4. Comprobar que se pase a la modalidad ELR cuando el cinturón está completamente desenrollado.

• En caso de funcionamiento incorrecto, sustituir el cinturón de seguridad.

Enrollador limitador de carga

Atención
• Con la intervención del limitador de carga, el cinturón y el anclaje se frotan entre si dejando un signo de desgaste. En estas condiciones el cinturón de seguridad podría no funcionar correctamente con consiguiente peligro de accidentes. Asegurarse de sustituir el cinturón de seguridad después de la intervención del limitador de carga.

1. En caso de accidente, tirar el cinturón del enrollador y asegurarse que no hay signos de desgaste (ninguna intervención de limitador de carga) controlando a vista y al tacto.

• En caso de funcionamiento incorrecto, sustituir el cinturón de seguridad.
