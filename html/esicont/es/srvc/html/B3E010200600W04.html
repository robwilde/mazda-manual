DTC P0668 [ZJ, Z6]

B3E010200600W04

DTC P0668

Entrada baja circuito sensor de temperatura PCM

CONDICCION DE DETECCION

• El PCM controla la señal del sensor de temperatura PCM (integrado en el PCM). Si el PCM señala que la tensión del sensor de temperatura PCM es inferior a 0,10 V, el PCM determina que el circuito del sensor de temperatura PCM no funciona correctamente.
Detalles de soporte de diagnóstico
• Esto es un control continuo (Otros).
• El MIL no se enciende.
• El CODIGO PENDIENTE está disponible si el PCM detecta el funcionamiento incorrecto sobre mencionado.
• Los DATOS PID INSTANTANEOS MEMORIZADOS no son disponibles.
• El DTC está registrado en la memoria del PCM.

PROBABLE CAUSA

• Funcionamiento incorrecto sensor de temperatura PCM
