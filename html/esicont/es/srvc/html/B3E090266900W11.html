DTC 03:Er01

Lector CD

CONDICCION DE DETECCION

• El lector de CD no se abre o cierra para introducir o quitar el disco.

PROBABLE CAUSA

• CD defectuoso (roto o contaminado por impurezas)
• Funcionamiento incorrecto lector CD

PASO

INSPECCION

ACCION

1
Comprobar las condiciones del CD cuando se produzca un error.
¿El CD está normal?
Sí
Pasar al paso siguiente.
No
Reparar (limpiar) o sustituir el CD y pasar al paso siguiente.
2
Borrar el DTC.
Introducir y luego quitar un CD normal.
¿Se visualiza el DTC 03:Er01?
Sí
Sustituir el lector de CD.
No
Búsqueda averías DTC completada.
