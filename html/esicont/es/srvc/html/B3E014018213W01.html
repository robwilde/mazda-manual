CONTROL SENSOR PRESION ABSOLUTA COLECTOR (MAP) (LF)

B3E014018213W01

Nota
• Antes de llevar a cabo el siguiente control, asegurarse de seguir el procedimiento indicado en el esquema de búsqueda averías.

Control tensión

1. Quitar el sensor MAP con el conector todavía conectado. [Véase REMOCION/INSTALACION SENSOR PRESION ABSOLUTA COLECTOR (MAP) (LF)].

2. Desconectar el tubo de vacío del sensor MAP.

3. Comprobar que la tensión en el terminal 2AL del PCM esté conforme a lo especificado cuando el conmutador de arranque está en posición ON.

• Si no está conforme a lo especificado, sustituir el sensor MAP.
Tensión
2,69-4,37 V

4. Instalar la bomba de vacío.

5. Poner el conmutador de arranque en posición ON.

6. Comprobar que el cambio de tensión esté conforme a lo especificado cuando se aplica un vacío de 30 kPa (225 mmHg; 8,86 inHg) con una bomba de vacío.

• Si no está conforme a lo especificado, sustituir el sensor MAP.
Variación de tensión
1,16-1,27 V

Control circuitos abiertos/cortocircuitos

1. Desconectar el conector del PCM. [Véase REMOCION/INSTALACION PCM (LF)].

2. Controlar si los siguientes cableados están abiertos o en cortocircuito. (Control de la continuidad)

Circuito abierto

• Si no hay continuidad, el circuito está abierto. Reparar o sustituir el cableado.
- Terminal A del sensor MAP y terminal 2AA del PCM
- Terminal D del sensor MAP y terminal 2AL del PCM
- Terminal C del sensor MAP y terminal 2W del PCM

Cortocircuito

• Si hay continuidad, el circuito está en cortocircuito. Reparar o sustituir el cableado.
- Terminal C del sensor MAP y alimentación.
- Terminal C del sensor MAP y masa carrocería.
- Terminal D del sensor MAP y alimentación.
- Terminal D del sensor MAP y masa carrocería.
