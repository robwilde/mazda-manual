CONDICCION DE DETECCION

Atención
• Las condiciones de detección sirven para comprender el DTC antes de efectuar un control. Efectuar una inspección sólo de acuerdo con las condiciones de detección puede causar lesiones debidas a un error de funcionamiento o daños al sistema. Cuando se efectue un control, seguir siempre el procedimiento de control.
• Funcionamiento incorrecto en el cableado entre el sensor airbag lateral lado conductor y el módulo de control SAS
• Funcionamiento incorrecto circuito sensor airbag lateral lado conductor

PROBABLE CAUSA

• Circuito abierto o cortocircuito en el cableado entre el sensor airbag lateral lado conductor y el módulo de control SAS
• Funcionamiento incorrecto sensor airbag lateral lado conductor
• Funcionamiento incorrecto del módulo de control SAS
