CONTROL SENSOR PRESION BAROMETRICA (BARO) (ZJ, Z6)

B3E014018210W03

Nota
• Antes de llevar a cabo el siguiente control, asegurarse de seguir el procedimiento indicado en el esquema de búsqueda averías. (Véase Procedimiento de búsqueda averías).

Control tensión

1. Quitar el sensor BARO con el conector todavía conectado.

2. Quitar el tubo flexible del sensor BARO.

3. Poner el conmutador de arranque en posición ON.

4. Comprobar que la tensión de salida del sensor BARO (PID WDS: BARO) esté conforme a lo especificado. [Véase CONTROL PCM (ZJ, Z6)].

• Si no está conforme a lo especificado, aunque el cableado no presente funcionamiento incorrecto, sustituir el sensor BARO.

5. Instalar la bomba de vacío al sensor BARO.

6. Comprobar que el cambio en la tensión de salida del sensor BARO (PID WDS: BARO) esté conforme a lo especificado cuando se aplica un vacío de 30 kPa (0,30 kgf/cm2; 4.4 psi).

• Si no está conforme a lo especificado, sustituir el sensor BARO.
Variación tensión de salida sensor BARO
0,5-0,7 V
Nota
• La tensión que se muestra en la figura puede variar excesivamente dependiendo del tiempo y de las condiciones de la batería.

Control circuitos abiertos/cortocircuitos

1. Quitar la tapa del conector del PCM.

2. Desconectar el conector del PCM. [Véase REMOCION/INSTALACION SISTEMA DE ADMISION (ZJ, Z6)].

3. Controlar por si hay circuito abierto o cortocircuito en los siguientes cableados (control continuidad).

Circuito abierto

• Si no hay continuidad, el circuito está abierto. Reparar o sustituir el cableado.
- Terminal C del sensor BARO y terminal 2W del PCM
- Terminal A del sensor BARO y el terminal 2AX del PCM
- Terminal B del sensor BARO y terminal 2S del PCM

Cortocircuito

• Si hay continuidad, hay un cortocircuito. Reparar o sustituir el cableado.
- Terminal C del sensor BARO y masa carrocería
- Terminal A del sensor BARO y alimentación
- Terminal B del sensor BARO y masa carrocería
- Terminal B del sensor BARO y alimentación
