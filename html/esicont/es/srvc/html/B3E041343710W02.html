CONTROL SENSOR TRASERO VELOCIDAD RUEDA ABS

B3E041343710W02

Control visual de instalación

1. Controlar lo siguiente:

• En caso de funcionamiento incorrecto, sustituir el componente.
(1) Aflojamiento o juego excesivo del sensor de velocidad rueda ABS
(2) Deformación del sensor de velocidad rueda ABS
(3) Deformación o daños del rotor del sensor ABS

Control juego

1. Quitar el sensor velocidad rueda ABS trasera.

2. Medir la distancia entre la superficie de instalación del sensor velocidad rueda ABS trasera y el rotor del sensor ABS. Esta es la dimensión A.

3. Calcular el juego entre el sensor velocidad rueda ABS trasera y el rotor del sensor ABS con la siguiente fórmula.

Juego [mm (in)] = A -13,9 (0,547)

4. Comprobar que el juego entre el rotor del sensor ABS y el sensor de velocidad rueda ABS trasera esté conforme a cuanto indicado a continuación.

• En caso de funcionamiento incorrecto, sustituirlo.
Juego
1,46 mm (0,057 in) o inferior

Control valor que sale del sensor

Advertencia
• Controlar la resistencia con otros comprobadores puede dañar el circuito interior del sensor velocidad rueda ABS. Asegurarse de utilizar el WDS o instrumento equivalente para controlar el sensor velocidad rueda ABS.

1. Poner el conmutador de arranque en posición OFF.

2. Conectar el WDS o instrumento equivalente al DLC-2.

3. Seleccionar los siguientes PID mediante el WDS o instrumento equivalente:

• LR_WSPD
(Sensor velocidad rueda trasera izquierda)
• RR_WSPD
(Sensor velocidad rueda trasera derecha)

4. Poner en marcha el motor y conducir el vehículo.

5. Comprobar que la pantalla del WDS o equivalente muestre el mismo valor que el taquímetro.

• En caso de funcionamiento incorrecto, sustituir el sensor de velocidad rueda ABS.
