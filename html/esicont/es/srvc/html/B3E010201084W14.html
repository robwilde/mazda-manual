DTC P0118

Entrada alta circuito sensor ECT

CONDICCION DE DETECCION

• El PCM controla la señal del sensor ECT en el terminal 2AK del PCM. Si el PCM detecta que la tensión del sensor ECT essuperior a 4,6 V, el PCM establece que el circuito del sensor ECT no funciona correctamente.
Detalles de soporte de diagnóstico
• Esto es un control continuo (CCM).
• El MIL se enciende si el PCM detecta condiciones de funcionamiento incorrecto en el primer ciclo de conducción.
• El CODIGO PENDIENTE está disponible si el PCM detecta el funcionamiento incorrecto sobre mencionado.
• Las INSTANTANEAS MEMORIZADAS son disponibles.
• El DTC está registrado en la memoria del PCM.

PROBABLE CAUSA

• Funcionamiento incorrecto del sensor ECT
• Circuito abierto en el cableado entre el terminal A del sensor ECT y el terminal 2AK del PCM
• Cortocircuito hacia alimentación en el cableado entre el terminal A del sensor ECT y el terminal 2AK del PCM
• Circuito abierto en el cableado entre el terminal B del sensor ECT y el terminal 2AA del PCM
• Conexión defectuosa del conector del sensor ECT o del PCM.
• Funcionamiento incorrecto del PCM
