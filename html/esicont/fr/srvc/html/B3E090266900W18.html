DTC 06 : Er02

Changeur de CD (module supérieur)

CONDITIONS DE DETECTION

Remarque
• Lorsque l'erreur survient, le message d'erreur 'CHECK CD' s'affiche sur l'écran LCD.
• Le CD ne peut pas être lu normalement ou les pistes ne peuvent pas être changées.

CAUSE POSSIBLE

• CD défectueux (rayures ou salissures)
• Anomalie du changeur de CD (module supérieur)

Procédure de diagnostic

ETAPE

INSPECTION

ACTION

1
Lire tous les CD se trouvant dans le changeur, en passant d'une plage à l'autre.
Est-il possible de passer d'un plage à l'autre ?
Oui
Passer à l'étape 3.
Non
Certaines plages ne peuvent être sélectionnées.
Passer à l'étape suivante.
Aucune plage ne peut être sélectionnée.
Changer de CD et tenter à nouveau de passer d'une plage à l'autre.
• Si le fonctionnement est normal, passer à l'étape 3.
• Si le fonctionnement est anormal, remplacer le changeur de CD (module supérieur).
2
Contrôler l'état du CD lorsque l'erreur apparaît.
Le CD est-il en bon état ?
Oui
Passer à l'étape suivante.
Non
Réparer (retirer les corps étrangers, nettoyer) ou remplacer le CD, puis passer à l'étape suivante.
3
Effacer le DTC.
Lire le CD et passer d'un plage à l'autre.
Le DTC 06 : Er02 s'affiche-t-il ?
Oui
Remplacer le changeur de CD (module supérieur).
Non
Dépistage des pannes à partir des DTC terminé.
