PROCEDURE DE DEMARRAGE DE MODE DE TEST DE DIAGNOSTIC EMBARQUE

B3E090266900W01

Remarque
• Tous les DTC affichés dans le mode de test du diagnostic embarqué doivent être notés sur le formulaire de commande de réparation audio.

1. Positionner le contacteur d'allumage sur ACC ou ON.

2. Appuyer simultanément sur le bouton POWER, le bouton AM et le bouton CLOCK pendant au moins 2 secondes.

1
Bouton AM
2
Bouton CLOCK
3
Bouton SEEK (sans RDS)
4
Bouton SEEK (avec RDS)
5
Bouton POWER

Remarque
• Si plusieurs DTC sont en mémoire, on peut les afficher à l'aide du bouton SEEK.

3. Pour arrêter le mode de test du diagnostic embarqué, positionner le contacteur d'allumage sur LOCK.
