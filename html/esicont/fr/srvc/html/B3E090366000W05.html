PROCEDURE N° 4, LE SYSTEME NE CHARGE PAS LE DISQUE DE CARTE

B3E090366000W05

4

Le système ne charge pas le disque de carte.

CAUSE POSSIBLE

• Anomalie de l'unité de navigation du véhicule
• Anomalie du disque de carte
• Version de disque d'application incorrecte
• Humidité dans l'unité de navigation du véhicule
• Disque inséré inapproprié
Remarque
• Si une version d'application plus récente est insérée, l'unité de navigation du véhicule est automatiquement mise à jour. Par conséquent, le disque de carte précédent devient inutilisable.
• Si de la buée se forme dans l'unité de navigation du véhicule, l'écran affiche le message 'Cannot Read Disc' (impossible de lire le disque). Retirer le disque de carte, aérer ou éliminer l'humidité du compartiment passager, et le laisser sécher pendant approx. 1 h. La buée sera éliminée, et l'unité de navigation du véhicule fonctionnera à nouveau correctement.
• En cas de disque de carte sale/endommagé ou d'unité de navigation du véhicule dysfonctionnelle, l'écran affiche le message "Accessing Data' (accès aux données) pendant 1 à 2 minutes, puis le message 'Cannot Read Disc" (impossible de lire le disque) après que le contacteur d'allumage ait été positionné sur ACC. Cependant, si une ancienne version du disque de carte est insérée, l'écran affiche le message 'Cannot Read Disc" (impossible de lire le disque) dès que le contacteur d'allumage est positionné sur ACC.
• Si un disque inapproprié est inséré, l'écran affiche le message 'Cannot Read Disc" (impossible de lire le disque).
