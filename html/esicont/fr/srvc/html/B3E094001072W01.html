DEPOSE/REPOSE DE BOITE DE RACCORDEMENT COTE PASSAGER (PJB)

B3E094001072W01

Attention
• Lors du remplacement de PJB, la procédure de configuration doit être effectuée avant de déposer le PJB. Le fait de remplacer le PJB sans avoir effectué la procédure préalable de configuration provoquera un dysfonctionnement du système.

1. Effectuer la configuration de PJB. (voir la section CONFIGURATION DE BOITE DE RACCORDEMENT COTE PASSAGER (PJB).)

2. Débrancher le câble négatif de la batterie.

3. Déposer les différents éléments selon l'ordre indiqué dans le tableau.

.

1
Couvercle
2
Connecteur A
(voir la section Note sur la dépose de connecteur A.)
(voir la section Note sur la repose de connecteur A.)
3
Connecteur B
(voir la section Note sur la dépose de connecteur B.)
(voir la section Note sur la repose de connecteur B.)
4
PJB
(voir la section Note sur la dépose de PJB.)

4. Pour la repose, suivre l'ordre inverse de la dépose.

Note sur la dépose de connecteur A

1. Pousser la languette de déblocage dans le sens indiqué par la flèche.

2. Tourner le levier dans le sens indiqué par la flèche, et déposer le connecteur A.

Note sur la dépose de connecteur B

1. Tourner le levier dans le sens indiqué par la flèche, et déposer le connecteur B.

Note sur la dépose de PJB

1. Tourner les vis dans le sens antihorloger, afin de déposer le PJB.

Remarque
• Les vis ne peuvent être retirées du PJB.

2. Déposer le PJB, comme indiqué dans l'illustration.

Note sur la repose de connecteur B

1. Après avoir rebranché le connecteur, tourner le levier dans le sens indiqué par la flèche afin de reposer le connecteur B.

Note sur la repose de connecteur A

1. Après avoir rebranché le connecteur, tourner le levier dans le sens indiqué par la flèche afin de reposer le connecteur A.
