INSPECTION DE CONTACTEUR DE PRESSION D'HUILE

B3E092218500W01

1. Vérifier que le témoin de pression d'huile s'allume lorsque le contacteur d'allumage est positionné sur ON.

2. S'assurer que le témoin de pression d'huile s'éteint lorsque le moteur démarre.

• Si le témoin de pression d'huile ne s'allume pas ou s'il reste allumé, inspecter le faisceau de câblage correspondant.
- Si le faisceau de câblage connexe est normal, inspecter la pression d'huile. (voir les sections INSPECTION DE PRESSION D'HUILE [LF], INSPECTION DE PRESSION D'HUILE [ZJ, Z6].)
• Si la pression d'huile est normale, remplacer le contacteur de pression d'huile.
