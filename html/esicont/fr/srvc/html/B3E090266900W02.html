PROCEDURE D'IDENTIFICATION DE FOURNISSEUR

B3E090266900W02

Remarque
• Le fournisseur peut varier en fonction du module. Lors d'une recherche de fournisseur (centre d'entretien) pour une réparation ou un remplacement, identifier le fournisseur et remplir le formulaire de commande de réparation audio en suivant les procédures suivantes.

Identification via la marque ou l'appellation inscrite

1. Déposer l'unité audio.

2. Vérifier le fournisseur en se reportant à l'étiquette accompagnant chaque unité.

Identification via le mode test de diagnostic embarqué.

1. Lancer le mode test de diagnostic embarqué.

2. Identifier l'équipement et le fournisseur en se reportant à l'affichage LCD.

Remarque
• Si aucun DTC n'est enregistré, aucun code ne s'affichera.
