DTC P0101 [LF]

B3E010201084W05

DTC P0101

Problème de plage/performances de circuit de capteur MAF

CONDITIONS DE DETECTION

• Le PCM compare la valeur MAF en cours à la valeur MAF prévue lorsque le moteur tourne.
- Si la quantité de débit d'admission d'air massique est inférieure à 5,0 g/s {0,66 lb/min} pendant 5 secondes et que l'angle d'ouverture de papillon est supérieure à 50% lorsque le moteur tourne, le PCM détermine que la quantité de débit d'admission d'air massique est trop faible.
- Si la quantité de débit d'admission d'air massique est supérieure à 96,0 g/s {12,7 lb/min} pendant 5 secondes et que le régime moteur est inférieur à 2 000 tr/mn lorsque le moteur tourne, le PCM détermine que la quantité de débit d'admission d'air massique est trop élevée.
Note sur le support de diagnostic
• Il s'agit d'un contrôle continu (CCM).
• Le témoin MIL s'allume si le PCM détecte l'anomalie ci-dessus dans deux cycles de conduite consécutifs, ou un cycle de conduite alors que le DTC de la même anomalie a été enregistré dans le PCM.
• Le CODE EN COURS est disponible si le PCM détecte les conditions d'anomalie ci-dessus au cours du premier cycle de conduite.
• DONNEES ARRETEES disponibles.
• Le DTC est enregistré dans la mémoire du PCM.

CAUSE POSSIBLE

• Anomalie du capteur MAF
• Anomalie du capteur TP
• Corrosion électrique dans le circuit de signal du capteur MAF
• Corrosion électrique dans le circuit MAF RETURN
• Chute de tension dans le circuit de signal du capteur MAF
• Chute de tension dans le circuit de masse
