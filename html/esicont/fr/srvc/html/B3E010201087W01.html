DTC P0500

Problème du circuit VSS

CONDITIONS DE DETECTION

• L PCM contrôle la vitesse du véhicule à partir de l'HU/CM d'ABS ou de l'HU/CM de DSC. Si le PCM ne reçoit pas le signal d'entrée de vitesse du véhicule, il détermine una anomalie du circuit VSS.
Note sur le support de diagnostic
• Il s'agit d'un contrôle continu (CCM).
• Le témoin MIL s'allume si le PCM détecte l'anomalie ci-dessus dans deux cycles de conduite consécutifs, ou un cycle de conduite alors que le DTC de la même anomalie a été enregistré dans le PCM.
• Le CODE EN COURS est disponible si le PCM détecte la condition d'anomalie ci-dessus au cours du premier cycle de conduite.
• DONNEES ARRETEES disponibles.
• Le DTC est enregistré dans la mémoire du PCM.

CAUSE POSSIBLE

• Anomalie du circuit VSS
