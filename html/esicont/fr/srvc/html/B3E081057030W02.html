CONFIGURATION DE MODULE DE COMMANDE SAS

B3E081057030W02

1. Connecter le WDS ou équivalent au DLC-2 (16 broches).

2. Sélectionner 'Programmation de module' dans le menu.

3. Sélectionner 'Installation de module programmable'.

4. Sélectionner les éléments suivants, et effectuer la procédure en suivant les instructions qui s'affichent sur l'écran WDS ou équivalent.
