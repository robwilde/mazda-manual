CODE OBD DE PROBLEME EN COURS [LF]

B3E010200102W03

• Ils apparaissent lorsqu'un problème est détecté dans un système contrôlé. Le code pour un système défaillant est mémorisé dans la mémoire du PCM pendant le premier cycle de conduite. Ce code est appelé code en cours. Si le PCM juge que le système est retourné à la normale ou que le problème détecté est une erreur, il supprime le code en cours. Si le problème est localisé également pendant le deuxième cycle de conduite, le PCM estime que le système est défaillant, et il mémorise le DTC.
