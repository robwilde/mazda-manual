DTC B1274, B1275

Système d'actionneur de mode de débit d'air (potentiomètre)

CAUSE POSSIBLE

• Anomalie de l'actionneur de mode de débit d'air
• Circuit ouvert dans le faisceau de câblage entre l'unité de commande de climatisation et l'actionneur de mode de débit d'air
• Court-circuit dans le faisceau de câblage entre l'unité de commande de climatisation (borne L) et l'actionneur de mélange d'air (borne C)

Procédure de diagnostic

ETAPE

INSPECTION

ACTION

1
• Inspecter l'actionneur de mode de débit d'air. (voir la section INSPECTION D'ACTIONNEUR DE MODE DE DEBIT D'AIR.)
• Fonctionne-t-il normalement ?
Oui
Passer à l'étape suivante.
Non
Remplacer l'actionneur de mode de débit d'air. (voir la section DÉPOSE/REPOSE D'ACTIONNEUR DE MODE DE DÉBIT D'AIR.)
2
• Débrancher le connecteur de l'unité de commande de climatisation et le connecteur de l'actionneur de mode de débit d'air.
• Y a-t-il circuit ouvert dans le faisceau de câblage entre les bornes suivantes de l'unité de commande de climatisation et l'actionneur de mode de débit d'air ?
- P-B
- L-C
- U-A
Oui
Réparer le faisceau de câblage.
Non
Passer à l'étape suivante.
3
• Y a-t-il court-circuit à la masse dans le faisceau de câblage entre la borne L de l'unité de commande de climatisation et la borne C de l'actionneur de mode de débit d'air ?
Oui
Réparer le faisceau de câblage.
Non
Le système est actuellement en bon état. (effacer l'anomalie antérieure de la mémoire.)
