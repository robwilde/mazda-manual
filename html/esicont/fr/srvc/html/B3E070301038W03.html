PROCEDURE N° 1, INSUFFISANCE D'AIR (OU PAS D'AIR) SOUFFLE PAR LES BOUCHES D'AERATION

B3E070301038W03

1

Les bouches d'aération soufflent une quantité d'air insuffisante (ou nulle)

DESCRIPTION

• Chaque bouche et/ou conduite présente ce problème.

CAUSE POSSIBLE

• Anomalie du système de mode VENTILATION (étapes 1-4)
• Anomalie du système de mode CHAUFFAGE (étape 5)
• Anomalie du système de mode DEGIVRAGE (étapes 6-8)

Procédure de diagnostic

ETAPE

INSPECTION

ACTION

1

VERIFIER SI L'ANOMALIE SE SITUE DANS LE MODE VENTILATION OU DANS UN AUTRE MODE

• De l'air est-il soufflé quand le mode VENTILATION est sélectionné ?
Oui
Passer à l'étape 4.
Non
Passer à l'étape suivante.
2

VERIFIER LA BOUCHE D'AERATION

• La bouche d'aération est-elle obstruée ?
Oui
La déboucher et passer à l'étape 8.
Non
Passer à l'étape suivante.
3

VERIFIER QUE LE CONDUIT DU TABLEAU DE BORD EST INSTALLE

• Le conduit du tableau de bord est-il correctement monté ?
Oui
Contrôler le conduit pour détecter les obstructions, les déformations et les fuites d'air, puis passer à l'étape 8.
Non
Reposer solidement le conduit dans la position adéquate, puis passer à l'étape 8.
4

VERIFIER SI L'ANOMALIE SE SITUE DANS LE MODE CHAUFFAGE OU DANS LE MODE DEGIVRAGE

• De l'air est-il soufflé quand le mode CHAUFFAGE est sélectionné ?
Oui
Passer à l'étape suivante.
Non
Vérifier si la bouche d'aération est obstruée, puis passer à l'étape 8.
5

INSPECTER LE MODE DEGIVRAGE

• De l'air est-il soufflé quand le mode DEGIVRAGE est sélectionné ?
Oui
Le fonctionnement est normal. Revérifier les symptômes d'anomalie.
Non
Passer à l'étape suivante.
6

VERIFIER LA BOUCHE D'AERATION

• La bouche d'aération est-elle obstruée ?
Oui
La déboucher et passer à l'étape 8.
Non
Passer à l'étape suivante.
7

VERIFIER QUE LE CONDUIT DE DEGIVRAGE EST INSTALLE

• Le conduit de dégivrage est-il correctement monté ?
Oui
Contrôler le conduit pour détecter les obstructions, les déformations et les fuites d'air, puis passer à l'étape 8.
Non
Reposer solidement le conduit dans la position adéquate, puis passer à l'étape suivante.
8

CONFIRMER QUE LE SYMPTOME D'ANOMALIE NE PERSISTE PAS APRES LA REPARATION

• De l'air est-il expulsé ?
Oui
Dépistage des pannes terminé.
Informer le client des réparations exécutées.
Non
Revérifier les symptômes d'anomalie, puis répéter la procédure à partir de l'étape 1 si l'anomalie réapparaît.
