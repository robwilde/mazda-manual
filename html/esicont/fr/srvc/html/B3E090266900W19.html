DTC 06 : Er07

Changeur de CD (module supérieur)

CONDITIONS DE DETECTION

Remarque
• Lorsque l'erreur survient, le message d'erreur 'CHECK CD' s'affiche sur l'écran LCD.
• L'unité de base détecte une erreur de lecture de CD.

CAUSE POSSIBLE

• CD défectueux (rayures ou salissures)
• Anomalie du changeur de CD (module supérieur)

Procédure de diagnostic

ETAPE

INSPECTION

ACTION

1
Lire tous les CD se trouvant dans le changeur.
Leur lecture est-elle normale ?
Oui
Passer à l'étape 3.
Non
Certains CD ne peuvent être lus.
Passer à l'étape suivante.
Aucun CD ne peut être lu.
Passer à l'étape 3.
2
Contrôler l'état du CD lorsque l'erreur apparaît.
Le CD est-il en bon état ?
Oui
Passer à l'étape suivante.
Non
Réparer (retirer les corps étrangers, nettoyer) ou remplacer le CD, puis passer à l'étape suivante.
3
Effacer le DTC, puis insérer un CD en bon état.
Lire le CD pendant au moins 12 secondes.
Le DTC 06 : Er07 s'affiche-t-il ?
Oui
Remplacer le changeur de CD (module supérieur).
Non
Dépistage des pannes à partir des DTC terminé.
