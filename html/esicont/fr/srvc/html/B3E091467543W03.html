ENREGISTREMENT DE CODE D'IDENTIFICATION DE TÉLÉCOMMANDE.

B3E091467543W03

Remarque
• Lors de l'enregistrement du code d'identification dans une unité de télécommande, vérifier que d'autres télécommandes ne fonctionnement pas à proximité.
• Une fois terminé l'enregistrement du code d'identification, retirer la clé de l'antivol de direction et vérifier que toutes les portes se verrouillent/déverrouillent normalement en utilisant la télécommande.
