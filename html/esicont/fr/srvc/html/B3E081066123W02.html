REGLAGE DE RESSORT HELICOIDAL

B3E081066123W02

Remarque
• La procédure de réglage est également indiquée sur l'étiquette de précautions située sur le ressort hélicoïdal.

1. Orienter les pneus avant en position droite.

Attention
• Le ressort hélicoïdal cassera s'il est trop enroulé. Ne pas forcer en tournant le ressort hélicoïdal.

2. Tourner le ressort hélicoïdal dans le sens horloger jusqu'à ce qu'il s'arrête.

3. À partir de la position d'arrêt, tourner le ressort hélicoïdal dans le sens antihorloger de 2 fois 3/4 de tours.

4. Aligner les repères.
