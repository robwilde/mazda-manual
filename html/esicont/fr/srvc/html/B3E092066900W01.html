SYSTEME ANTIVOL

B3E092066900W01

Conditions d'alarme

• Si la protection du système antivol a été activée, l'une quelconque des conditions suivantes déclenche le système :

- Un câble de batterie débranché
- Une batterie déchargée
- Des connecteurs de l'unité audio débranchés

• Si le système est déclenché, l'unité audio sera inopérationnelle lorsqu'elle est rebranchée sur une source d'alimentation et « code » clignote sur l'affichage jusqu'à ce que le numéro de code présélectionné ait été entré. Si le système antivol est déclenché, suivre les procédures appartenant à « L'annulation du fonctionnement antivol » pour réinitialiser l'unité.

Définition du numéro de code

Procéder à chaque étape (étapes 1-3) dans les 10 secondes ; dans le cas contraire, la procédure de réglage sera annulée.

1. Positionner le contacteur d'allumage sur ACC, et éteindre l'unité audio.

2. Appuyer d'abord sur le bouton AUTO-M, puis simultanément sur le bouton du canal 6 pendant environ 2 secondes jusqu'à ce que 'CODE' apparaisse sur l'affichage.

3. Appuyer à nouveau sur le bouton AUTO-M, puis simultanément sur le bouton du canal 6, jusqu'à ce que des tirets apparaissent sur l'affichage.

4. Sélectionner un numéro de code personnel, et le mémoriser avant de l'introduire. Si le numéro est introduit, puis oublié, il ne peut être supprimé, et l'unité est à nouveau déconnectée, l'unité audio étant dès lors inopérante.

5. Utiliser les bouton de canal 1-4 pour entrer le numéro de code sélectionné. Appuyer sur le bouton 1 pour le premier chiffre, 2 pour le deuxième, 3 pour le troisième et 4 pour le dernier chiffre. Introduire le numéro dans les 10 secondes. Si l'affichage est supprimé, répéter la procédure à partir de l'étape 1.

Remarque
• Effectuer l'étape 6 dans les 10 secondes suivant l'étape 5.

6. Le numéro de code étant affiché, appuyer sur le bouton AUTO-M, puis simultanément sur le bouton de canal 6 pendant environ 2 secondes. 'CODE' s'affiche pendant environ 5 secondes. Lorsque cet affichage a disparu, le numéro de code est défini.

7. Si « Err » (erreur) apparaît sur l'affichage, répéter la procédure à partir de l'étape 1. Si l'erreur d'entrée est répétée trois fois, positionner le contacteur d'allumage sur LOCK et répéter la procédure à partir de l'étape 1.

Annulation du numéro de code

Procéder à chaque étape (étapes 1-3) dans les 10 secondes ; dans le cas contraire, la procédure d'annulation sera annulée.

1. Positionner le contacteur d'allumage sur ACC, et éteindre l'unité audio.

2. Appuyer d'abord sur le bouton AUTO-M, puis simultanément sur le bouton du canal 6 pendant environ 2 secondes jusqu'à ce que 'CODE' apparaisse sur l'affichage.

3. Appuyer à nouveau sur le bouton AUTO-M, puis simultanément sur le bouton du canal 6, jusqu'à ce que des tirets apparaissent sur l'affichage.

4. Utiliser les boutons de canal 1-4 pour introduire le numéro de code actuel. Appuyer sur le bouton 1 pour le premier chiffre, 2 pour le deuxième, 3 pour le troisième et 4 pour le dernier chiffre. Introduire le numéro dans les 10 secondes. Si l'affichage est supprimé, répéter la procédure à partir de l'étape 1.

Remarque
• Effectuer l'étape 5 dans les 10 secondes suivant l'étape 4.

5. Le numéro de code étant affiché, appuyer sur le bouton AUTO-M, puis simultanément sur le bouton de canal 6 pendant environ 2 secondes. 'CODE' s'affiche pendant environ 5 secondes. Lorsque cet affichage a disparu, le numéro de code est annulé.

6. Si « Err » (erreur) apparaît sur l'affichage, répéter la procédure à partir de l'étape 4.

Annulation du fonctionnement antivol

Entrer correctement le numéro de code sélectionné pour désactiver le système antivol et reprendre le fonctionnement audio normal.

1. Positionner le contacteur d'allumage sur ACC. ('CODE' clignotera sur l'affichage.)

2. Appuyer d'abord sur le bouton AUTO-M, puis simultanément sur le bouton de canal 6 pendant environ 2 secondes, jusqu'à ce que des tirets apparaissent sur l'affichage.

Remarque
• Effectuer l'étape 3 dans les 10 secondes suivant l'étape 2.

3. Utiliser les bouton de canal 1-4 pour entrer le numéro de code sélectionné. Appuyer sur le bouton 1 pour le premier chiffre, 2 pour le deuxième, 3 pour le troisième et 4 pour le dernier chiffre. Introduire le numéro dans les 10 secondes. Si l'affichage est supprimé, répéter la procédure à partir de l'étape 1.

Attention
• Trois erreurs consécutives, y compris le fait de positionner le contacteur d'allumage sur LOCK et de débrancher l'unité audio, activeront le système antivol et rendront l'unité audio complètement inopérationnelle.

4. Le numéro de code étant affiché, appuyer sur le bouton AUTO-M, puis simultanément sur le bouton de canal 6 pendant environ 2 secondes. 'CODE' clignotera pendant environ 5 secondes.

5. Si « Err » (erreur) apparaît sur l'affichage, répéter la procédure à partir de l'étape 1.
