DTC B2141 [DSC]

B3E040243000W31

Remarque
• Le témoin de DSC s'allume et demeure allumé, et le DTC B2141 (donnée de véhicule non enregistrée) est enregistré dans la mémoire uniquement lorsque le contacteur d'allumage est positionné sur ON pour la première fois après le remplacement de l'HU/CM de DSC. Les données du véhicule étant téléchargées depuis le PCM via le système CAN lorsque le contacteur d'allumage est positionné sur ON, le DTC B2141 est enregistré uniquement en tant qu'anomalie passée lorsque le contacteur d'allumage est positionné sur OFF, puis à nouveau sur ON.

DTC

B2141

Données de véhicule non enregistrées

CONDITIONS DE DETECTION

• Les données du véhicule ne sont pas enregistrées dans l'HU/CM de DSC.

CAUSE POSSIBLE

• Il y a une anomalie dans le système : les données du véhicule n'ont pas été enregistrées dans le CM après le remplacement de l'HU/CM de DSC.
