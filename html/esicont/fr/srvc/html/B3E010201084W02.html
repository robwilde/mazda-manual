DTC P0032

Entrée haute du circuit de chauffage d'HO2S avant

CONDITIONS DE DETECTION

• Le PCM contrôle le signal de commande de chauffage d'HO2S avant au niveau de la borne 2G du PCM. Si le PCM allume l'élément chauffant de l'HO2S avant, mais que le circuit de l'élément chauffant d'HO2S avant présente une tension haute, le PCM détermine une anomalie du circuit de chauffage d'HO2S avant.
Remarque
• Le chauffage d'HO2S avant est contrôlé par un signal opératoire.
Note sur le support de diagnostic
• Il s'agit d'un contrôle continu (élément chauffant HO2S).
• Le témoin MIL s'allume si le PCM détecte l'anomalie ci-dessus dans deux cycles de conduite consécutifs, ou un cycle de conduite alors que le DTC de la même anomalie a été enregistré dans le PCM.
• LE CODE EN COURS est disponible si le PCM détecte la condition d'anomalie ci-dessus.
• DONNEES ARRETEES disponibles.
• Le DTC est enregistré dans la mémoire du PCM.

CAUSE POSSIBLE

• Court-circuit à l'alimentation dans le faisceau de câblage entre la borne D de l'HO2S avant et la borne 2G du PCM
• Court-circuit dans l'HO2S avant ou la borne de PCM
• Anomalie de l'élément chauffant d'HO2S avant
• Anomalie du PCM
