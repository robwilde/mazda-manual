Compteur de vitesse

A l'aide du mode de vérification entrée/sortie

1. Inspecter le compteur de vitesse en le réglant sur le code 12 du mode de vérification d'entrée/sortie. (voir la section MODE DE VERIFICATION D'ENTREE/SORTIE DE COMBINE D'INSTRUMENTS.)

A l'aide d'un testeur de compteur de vitesse

1. Régler la pression des pneus en fonction des valeurs spécifiées.

2. A l'aide d'un testeur de compteur de vitesse, vérifier que les valeurs lues par le testeur correspondent aux valeurs du tableau ci-après.

3. Vérifier que les valeurs lues par le compteur de vitesse correspondent à la plage de valeurs spécifiées dans le tableau.

• Si le compteur de vitesse ne se déplace pas ou que l'indication se situe en dehors de la plage de valeurs autorisées, inspecter le PCM et les faisceaux de câblage connexes.
- Si le PCM et les faisceaux de câblage connexes sont normaux, remplacer le combiné d'instruments.

Compte-tours

A l'aide du mode de vérification entrée/sortie

1. Inspecter le compte-tours en le réglant sur le code 13 du mode de vérification d'entrée/sortie. (voir la section MODE DE VERIFICATION D'ENTREE/SORTIE DE COMBINE D'INSTRUMENTS.)

À l'aide du WDS ou d'un équipement de diagnostic externe

Attention
• Si le régime du moteur dépasse les limites autorisées, le moteur pourrait être endommagé. Ainsi, lors de l'inspection du compte-tours, ne pas laisser le moteur dépasser le régime permis sur le compte-tours.

1. Brancher le WDS ou équivalent sur le connecteur 2 de diagnostic (16 broches).

2. Comparer l'élément de surveillance des donnees (RPM) avec les indications du compte-tours.

• Si le compte-tours ne fonctionne pas correctement, inspecter le PCM et les faisceaux de câblage connexes.
- Si le PCM et les faisceaux de câblage connexes ne présnetent aucune anomalie, remplacer le combiné d'instruments.

Jauge de carburant

1. Inspecter la jauge de carburant en le réglant sur le code 23 du mode de vérification d'entrée/sortie. (voir la section MODE DE VERIFICATION D'ENTREE/SORTIE DE COMBINE D'INSTRUMENTS.)

Jauge de température d'eau

1. Inspecter la jauge de température d'eau en le réglant sur le code 25 du mode de vérification d'entrée/sortie. (voir la section MODE DE VERIFICATION D'ENTREE/SORTIE DE COMBINE D'INSTRUMENTS.)
