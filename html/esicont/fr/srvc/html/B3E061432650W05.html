CONFIGURATION DE MODULE DE COMMANDE DE DIRECTION ASSISTÉE ÉLECTRO-HYDRAULIQUE (EHPAS)

B3E061432650W05

1. Positionner le contacteur d'allumage sur OFF.

2. Brancher le WDS ou équivalent sur le DLC-2.

3. Entrer les informations concernant le véhicule en suivant les instructions de l'écran du WDS ou équivalent.

4. Sélectionner 'Programmation de module' sur le WDS ou menu équivalent.

5. Sélectionner 'Installation de module programmable'.

6. Sélectionner "EPS".

7. Effacer les DTC à l'aide du WDS ou équivalent, puis vérifier qu'aucun DTC n'est présent.
