FONCTIONNEMENT DES SOLÉNOÏDES DE CHANGEMENT DE VITESSE A, B, C (DE TYPE RAPPORT CYCLIQUE)

B3E051721100A04

OUVERTURE : Lorsque le courant électrique ne passe pas, le port d'alimentation (pression de canalisation) dans le solénoïde s'ouvre et est engagé avec le port de sortie (pression d'embrayage). Ainsi, la pression hydraulique est fournie au passage hydraulique pour la pression d'embrayage.

FERMETURE : Lorsque le courant électrique passe, le port d'alimentation (pression de canalisation) dans le solénoïde se ferme, et le port de sortie (pression d'embrayage) et le port de purge sont engagés pour drainer la pression d'embrayage.
