Note sur le démontage de dispositif de retenue d'embrayage unidirectionnel

• Déposer le dispositif de retenue de l'embrayage unidirectionnel à l'aide d'un tournevis à tête plate, etc., comme indiqué dans l'illustration.

Procédure de remontage

1. Reposer le jonc d'arrêt.

2. Monter l'embrayage unidirectionnel sur le pignon intérieur avant dans le sens indiqué par la flèche (située sur l'embrayage unidirectionnel), comme indiqué dans l'illustration.

3. Monter la cage latérale.

4. Monter le dispositif de retenue de l'embrayage unidirectionnel.
