TIJDELIJKE STORINGSCODES [LF]

B3E010200102W03

• Deze worden weergegeven als er een probleem wordt waargenomen in een controlesysteem. De code voor het systeem waarin de fout zich voordeed, wordt tijdens de eerste rit opgeslagen in het geheugen van de PCM. Deze code wordt een tijdelijke storingscode genoemd. Als de PCM oordeelt dat het systeem weer in orde is of de storing per ongeluk is gesignaleerd, wordt de tijdelijke storingscode gewist. Als de storing ook tijdens de tweede rit waargenomen wordt, stelt de PCM vast dat er een storing is opgetreden in het systeem. De PCM slaat dan de storingscode op.
